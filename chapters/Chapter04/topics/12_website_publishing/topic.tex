\begin{learningbox}

\begin{itemize}
\item domain, hosting, URL ও web server ব্যাখ্যা করতে।
\item website publish করার ধাপ লিখতে।

\end{itemize}

\subsection{Domain ও Hosting}
ওয়েবসাইট তৈরি করার পর সেটি internet-এ দেখা যাওয়ার জন্য server-এ রাখতে হয়। ব্যবহারকারী domain name বা URL লিখে সেই server থেকে page দেখতে পায়।

\begin{definitionbox}{Domain name}
Internet-এ কোনো website-কে সহজে চেনার জন্য ব্যবহৃত human-readable address-কে domain name বলা হয়। যেমন \texttt{example.com}।
\end{definitionbox}

\begin{definitionbox}{Web hosting}
ওয়েবসাইটের file, image, database ইত্যাদি internet-connected server-এ সংরক্ষণ করে ব্যবহারকারীর কাছে পৌঁছে দেওয়ার সেবাকে web hosting বলা হয়।
\end{definitionbox}

\begin{keypointbox}{Publish checklist}
\begin{itemize}
\item Home page file সাধারণত \texttt{index.html} নাম রাখা।
\item সব link, image path ও form action test করা।
\item mobile view ও browser compatibility দেখা।
\item domain DNS ঠিকভাবে hosting server-এর দিকে point করা।
\item security-এর জন্য HTTPS ব্যবহার করা।
\end{itemize}
\end{keypointbox}
