\section{Website Publishing ও Hosting}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী পারবে:
\begin{itemize}
\item domain, hosting, URL, DNS ও web server ব্যাখ্যা করতে।
\item website publishing কী তা সংজ্ঞাসহ লিখতে।
\item website publish করার ধাপ লিখতে।
\item FTP/SFTP, control panel ও static hosting-এর ধারণা বুঝতে।
\item website publish করার পর common problem শনাক্ত করতে।
\end{itemize}
\end{learningbox}

\subsection{Website Publishing কী?}

\begin{definitionbox}{Website Publishing}
তৈরি করা webpage বা website-এর file web server-এ upload করে domain/URL-এর মাধ্যমে internet ব্যবহারকারীদের জন্য দেখার উপযোগী করার প্রক্রিয়াকে website publishing বলে।
\end{definitionbox}

Local computer-এ \texttt{index.html} file open করলে শুধু নিজের device থেকে website দেখা যায়। কিন্তু website publish করলে internet-এর মাধ্যমে অন্য user-ও URL লিখে website দেখতে পারে।

\subsection{Domain ও Hosting}

Website তৈরি করার পর সেটি internet-এ দেখানোর জন্য server-এ রাখতে হয়। ব্যবহারকারী domain name বা URL লিখে সেই server থেকে page দেখতে পায়।

\begin{definitionbox}{Domain name}
Internet-এ কোনো website-কে সহজে চেনার জন্য ব্যবহৃত human-readable address-কে domain name বলে। যেমন \texttt{example.com}।
\end{definitionbox}

\begin{definitionbox}{Web hosting}
Website-এর file, image, CSS, JavaScript, database ইত্যাদি internet-connected server-এ সংরক্ষণ করে user-এর কাছে পৌঁছে দেওয়ার সেবাকে web hosting বলে।
\end{definitionbox}

\subsection{Domain, Hosting, URL ও DNS-এর সম্পর্ক}

\begin{keypointbox}{সহজ ব্যাখ্যা}
\begin{itemize}
\item \textbf{Domain name} হলো website-এর নাম/ঠিকানা।
\item \textbf{Hosting} হলো যেখানে website-এর file রাখা হয়।
\item \textbf{URL} হলো নির্দিষ্ট webpage বা resource-এর পূর্ণ address।
\item \textbf{DNS} domain name-কে server IP address-এর সাথে map করে।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{Example}
\begin{itemize}
\item Domain: \texttt{abccollege.edu.bd}
\item Hosting: যে server-এ \texttt{index.html}, image, CSS file রাখা আছে
\item URL: \texttt{https://abccollege.edu.bd/notice.html}
\end{itemize}
\end{examplebox}

\subsection{Publish Checklist}

\begin{keypointbox}{Publish checklist}
\begin{itemize}
\item Home page file সাধারণত \texttt{index.html} নাম রাখা।
\item সব link, image path ও form action test করা।
\item mobile view ও browser compatibility দেখা।
\item domain DNS ঠিকভাবে hosting server-এর দিকে point করা।
\item security-এর জন্য HTTPS ব্যবহার করা।
\item upload করার আগে file/folder structure ঠিক রাখা।
\end{itemize}
\end{keypointbox}

\subsection{Website দেখা না গেলে সম্ভাব্য কারণ}

Website publish করার পর কখনো কখনো URL লিখলেও website দেখা যায় না। সম্ভাব্য কারণগুলো হলো:

\begin{itemize}
\item Domain name-এর মেয়াদ শেষ হয়ে যাওয়া।
\item Hosting account-এর মেয়াদ শেষ বা suspend হওয়া।
\item Web server down থাকা।
\item DNS configuration ভুল হওয়া বা DNS update complete না হওয়া।
\item Bandwidth limit শেষ হয়ে যাওয়া।
\item \texttt{index.html} file upload না করা বা ভুল folder-এ upload করা।
\item HTML file, image path বা permission ভুল থাকা।
\item HTTPS/SSL certificate সমস্যা হওয়া।
\end{itemize}

\subsection{Website Security Basic}

Website publish করার পর basic security বজায় রাখা জরুরি।

\begin{itemize}
\item HTTPS/SSL certificate ব্যবহার করা।
\item Hosting control panel-এর strong password রাখা।
\item নিয়মিত backup রাখা।
\item অপ্রয়োজনীয় file public folder-এ না রাখা।
\item Form থাকলে user input validate করা।
\item Hosting panel, CMS বা script update রাখা।
\end{itemize}

\subsection{Quick Publish Steps}

\begin{examplebox}{Publish via FTP/SFTP}
\begin{verbatim}
1. Test locally (open index.html in browser)
2. Connect to host via SFTP/FTP client
3. Upload files to public_html or www directory
4. Point domain DNS to hosting server IP
5. Open URL in browser and test
\end{verbatim}
\end{examplebox}

\subsection{Problem Solution}

\begin{practicebox}
\textbf{Problem:} Rafi একটি college website তৈরি করেছে। তার computer-এ website ঠিক দেখা যায়, কিন্তু hosting server-এ upload করার পর image দেখা যাচ্ছে না এবং home page automatically open হচ্ছে না। সম্ভাব্য কারণ ও সমাধান লেখ।

\textbf{Solution:}
\begin{itemize}
\item Home page automatically open না হলে file name \texttt{index.html} আছে কি না check করতে হবে।
\item Image দেখা না গেলে image file upload হয়েছে কি না check করতে হবে।
\item Image path case-sensitive কি না check করতে হবে; যেমন \texttt{Campus.jpg} ও \texttt{campus.jpg} ভিন্ন হতে পারে।
\item HTML file ও image একই folder-এ না থাকলে correct folder path দিতে হবে।
\item Browser cache clear করে আবার test করা যেতে পারে।
\end{itemize}
\end{practicebox}

\begin{boardbox}{Exam focus}
Website publish করার ধাপ লিখতে বললে: file তৈরি, local test, domain/hosting নেওয়া, server-এ upload, DNS setup, browser test--এই flow লিখবে। Website না দেখা গেলে domain expiry, hosting expiry, server down, DNS error, bandwidth limit, wrong file path ও missing \texttt{index.html} উল্লেখ করা যায়।
\end{boardbox}

\begin{mistakebox}{সাধারণ ভুল}
\begin{itemize}
\item home page-এর file name \texttt{index.html} না রাখা।
\item image upload না করে শুধু HTML file upload করা।
\item local computer-এর absolute path HTML code-এ ব্যবহার করা।
\item domain কিনে hosting/DNS setup না করা।
\item DNS update হতে কিছু সময় লাগতে পারে--এটা না বোঝা।
\item HTTPS/SSL setup না করা।
\item publish করার আগে link, image, form ও mobile view test না করা।
\end{itemize}
\end{mistakebox}
