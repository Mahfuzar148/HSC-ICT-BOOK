\section{ব্যবহারিক অনুশীলন: Mini Project}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী পারবে:
\begin{itemize}
\item HTML ব্যবহার করে একটি simple static website-এর পরিকল্পনা করতে।
\item Home, About, Notice, Result ও Contact page-এর structure বুঝতে।
\item heading, paragraph, image, hyperlink, list, table ও form একসাথে ব্যবহার করতে।
\item file/folder structure ঠিক রেখে website project তৈরি করতে।
\item HSC practical-style web page design task সমাধান করতে।
\end{itemize}
\end{learningbox}

HSC ICT বইয়ে অনেক সময় ``Mini Project'' নামে আলাদা section না থাকলেও Chapter 4-এ web page designing, HTML tag, hyperlink, image, table, form এবং website publishing শেখানো হয়। তাই Chapter 4-এর শেষে একটি ব্যবহারিক mini project রাখা খুব উপযোগী। এটি নতুন theory topic নয়; বরং পুরো chapter-এর tag ও concept practice করার একটি practical exercise।

\begin{keypointbox}{Book placement}
এই অংশটি chapter শেষে \textbf{ব্যবহারিক অনুশীলন: Mini Project} বা \textbf{Chapter Project} নামে রাখা ভালো। এতে শিক্ষার্থী HTML tag মুখস্থ করার পাশাপাশি একটি বাস্তব web page তৈরি করার অভ্যাস পাবে।
\end{keypointbox}

\subsection{Project Title}

\begin{definitionbox}{Project}
\textbf{``আমার কলেজ'' ওয়েবসাইট তৈরি} - এটি একটি simple static website project, যেখানে HTML-এর basic tag ব্যবহার করে কয়েকটি page তৈরি করা হবে।
\end{definitionbox}

\subsection{Project Objective}

এই project শেষে শিক্ষার্থীরা HTML ব্যবহার করে একটি simple static website তৈরি করতে পারবে। Website-এ heading, paragraph, image, hyperlink, list, table এবং form ব্যবহার থাকবে।

\subsection{Project Type}

\begin{itemize}
\item Static Website
\item Local Web Page Project
\item HSC ICT Chapter 4 Practical Practice
\end{itemize}

\item অন্য page তৈরি করো: \texttt{about.html} ও \texttt{notice.html}

\subsection{Home Page Code Explanation}

\begin{itemize}
\item \texttt{<!DOCTYPE html>} browser-কে জানায় যে document-টি HTML5 standard অনুসরণ করে।
\item \texttt{<html>} থেকে \texttt{</html>} পর্যন্ত সম্পূর্ণ HTML document লেখা হয়েছে।
\item \texttt{<head>} অংশে \texttt{meta charset} ও \texttt{title} রাখা হয়েছে।
\item \texttt{<body bgcolor="\#F5F7FA">} page-এর background color নির্ধারণ করেছে।
\item প্রথম \texttt{<table>} header section তৈরি করেছে, যেখানে logo ও college name আছে।
\item \texttt{<img src="images/logo.png">} logo image দেখানোর জন্য ব্যবহৃত হয়েছে।
\item \texttt{<a href="about.html">} ধরনের link ব্যবহার করে navigation menu তৈরি হয়েছে।
\item \texttt{<center>} ও \texttt{align="center"} image ও text center-এ দেখাতে সাহায্য করেছে।
\item \texttt{<ul>} facilities list এবং \texttt{<ol>} important links list তৈরি করেছে।
\item শেষ \texttt{<table>} footer section তৈরি করেছে।
\end{itemize}

\subsection{About Page Code Explanation}

\item \texttt{<h1>} main page heading দেখিয়েছে এবং \texttt{<h2>} দিয়ে ছোট ছোট section heading তৈরি হয়েছে।
\item \texttt{<p align="justify">} paragraph text দুই পাশে সমানভাবে সাজাতে ব্যবহার করা হয়েছে।

\item নিচের \texttt{<a>} tag গুলো Back to Home, Notice ও Contact page-এ যাওয়ার internal link তৈরি করেছে।
\end{itemize}

HTML শেখার সবচেয়ে ভালো উপায় হলো নিজে web page তৈরি করা। Mini project-এর মাধ্যমে শিক্ষার্থী heading, paragraph, image, hyperlink, list, table ও form একসাথে ব্যবহার করতে পারে। ফলে Chapter 4-এর গুরুত্বপূর্ণ tag ও structure ব্যবহারিকভাবে practice হয়।
\end{boardbox}
