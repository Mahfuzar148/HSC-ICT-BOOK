\section{ওয়েবসাইটের কাঠামো}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী পারবে:
\begin{itemize}
\item home page, sub page, navigation ও menu-এর কাজ ব্যাখ্যা করতে।
\item linear, hierarchical, network ও hybrid structure আলাদা করতে।
\item website map বা sitemap আঁকার ধারণা বুঝতে।
\end{itemize}
\end{learningbox}

\subsection{কাঠামোর ধারণা}
ওয়েবসাইটে কোন page থেকে কোন page-এ যাওয়া যাবে, কোন page প্রধান এবং কোন page সহায়ক—এই সম্পর্কের বিন্যাসকে website structure বলা যায়। ভালো structure ব্যবহারকারীকে বিভ্রান্ত না করে দ্রুত প্রয়োজনীয় তথ্যের কাছে নিয়ে যায়।

\begin{definitionbox}{Home page}
কোনো website-এ প্রবেশ করলে প্রথম যে প্রধান page দেখা যায়, তাকে home page বলা হয়। এখান থেকেই সাধারণত গুরুত্বপূর্ণ page-গুলোর link দেওয়া থাকে।
\end{definitionbox}

\begin{center}
\fbox{\parbox{0.92\textwidth}{
\centering
\textbf{Website structure diagram placeholders}\\[4pt]
Linear: Page 1 $\rightarrow$ Page 2 $\rightarrow$ Page 3\\
Tree: Home $\rightarrow$ About, Notice, Contact, Gallery\\
Network: প্রতিটি page থেকে একাধিক page-এ cross-link\\
Hybrid: Tree + selected cross-link
}}
\end{center}

\subsection{কাঠামোর ধরন}
\begin{center}
\begin{tabular}{|p{0.22\textwidth}|p{0.34\textwidth}|p{0.30\textwidth}|}
\hline
\textbf{Structure} & \textbf{ধারণা} & \textbf{ব্যবহার} \\
\hline
Linear & একটির পর একটি page & tutorial, story, step-by-step lesson \\
\hline
Hierarchical & root page থেকে branch & school/college website \\
\hline
Network & pageগুলো পরস্পর linked & wiki, knowledge site \\
\hline
Hybrid & একাধিক structure-এর সমন্বয় & বড় বাস্তব website \\
\hline
\end{tabular}
\end{center}

\begin{examplebox}{College website sitemap}
Home page থেকে Notice, Routine, Admission, Gallery ও Contact page-এ link থাকবে। Notice page থেকে PDF download link এবং Contact page থেকে form রাখা যেতে পারে।
\end{examplebox}

\begin{mistakebox}{সাধারণ ভুল}
Menu আর website structure একই জিনিস নয়। Menu হলো navigation-এর দৃশ্যমান অংশ; structure হলো page-গুলোর সামগ্রিক সম্পর্ক।
\end{mistakebox}

\begin{practicebox}
\begin{enumerate}
\item Sitemap কী?
\item Tree structure কোন ধরনের website-এ সুবিধাজনক?
\item Linear ও network structure-এর পার্থক্য লেখ।
\end{enumerate}
\end{practicebox}
