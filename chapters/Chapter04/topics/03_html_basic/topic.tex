\section{HTML-এর মৌলিক বিষয়}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী পারবে:
\begin{itemize}
\item HTML-এর পূর্ণরূপ ও কাজ ব্যাখ্যা করতে।
\item HTML file, browser rendering ও basic document structure বুঝতে।
\item HTML, CSS ও JavaScript-এর ভূমিকা আলাদা করতে।
\end{itemize}
\end{learningbox}

\subsection{HTML কী}
HTML বা HyperText Markup Language হলো web page তৈরির markup language। এটি tag ব্যবহার করে browser-কে content-এর structure জানায়। HTML file সাধারণত \texttt{.html} বা \texttt{.htm} extension দিয়ে সংরক্ষণ করা হয়।

\begin{definitionbox}{HTML}
HTML হলো এমন markup language যার মাধ্যমে heading, paragraph, image, link, table, form ইত্যাদি ব্যবহার করে web page তৈরি করা যায়।
\end{definitionbox}

\begin{center}
\fbox{\parbox{0.9\textwidth}{
\textbf{Basic HTML skeleton placeholder}\\[4pt]
\texttt{<!DOCTYPE html>}\\
\texttt{<html>}\\
\texttt{<head><title>My Page</title></head>}\\
\texttt{<body><h1>Hello</h1><p>First page</p></body>}\\
\texttt{</html>}
}}
\end{center}

\subsection{HTML, CSS ও JavaScript}
\begin{center}
\begin{tabular}{|p{0.22\textwidth}|p{0.34\textwidth}|p{0.30\textwidth}|}
\hline
\textbf{Technology} & \textbf{কাজ} & \textbf{উদাহরণ} \\
\hline
HTML & content structure & heading, paragraph, table \\
\hline
CSS & presentation/style & color, layout, spacing \\
\hline
JavaScript & behavior/interactivity & validation, button click, animation \\
\hline
\end{tabular}

\begin{mistakebox}{সাধারণ ভুল}
HTML programming language নয়; এটি markup language। HTML structure দেয়, CSS style দেয়, JavaScript behavior যোগ করে।
\end{mistakebox}
