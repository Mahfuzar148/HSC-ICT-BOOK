\section{HTML-এর মৌলিক বিষয়}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী পারবে:
\begin{itemize}
\item HTML-এর পূর্ণরূপ ও কাজ ব্যাখ্যা করতে।
\item HTML ও World Wide Web-এর ইতিহাসে Tim Berners-Lee, Robert Cailliau, CERN, W3C ও WHATWG-এর অবদান সংক্ষেপে বলতে।
\item HTML file, browser rendering ও basic document structure বুঝতে।
\item HTML, CSS ও JavaScript-এর ভূমিকা আলাদা করতে।
\end{itemize}
\end{learningbox}

\subsection{HTML কী}
HTML বা HyperText Markup Language হলো web page তৈরির markup language। এটি tag ব্যবহার করে browser-কে content-এর structure জানায়। HTML file সাধারণত \texttt{.html} বা \texttt{.htm} extension দিয়ে সংরক্ষণ করা হয়।

\begin{definitionbox}{HTML}
HTML হলো এমন markup language যার মাধ্যমে heading, paragraph, image, link, table, form ইত্যাদি ব্যবহার করে web page তৈরি করা যায়।
\end{definitionbox}

\begin{keypointbox}{ইতিহাস জানি: HTML-এর জন্ম}
HTML-এর পূর্ণরূপ HyperText Markup Language। ওয়েবপেজের content গঠন করার জন্য HTML ব্যবহৃত হয়। HTML-এর ইতিহাস World Wide Web-এর ইতিহাসের সাথে সরাসরি সম্পর্কিত।

১৯৮৯ সালে ব্রিটিশ বিজ্ঞানী Tim Berners-Lee CERN-এ কাজ করার সময় World Wide Web-এর ধারণা প্রস্তাব করেন। তাঁর লক্ষ্য ছিল বিভিন্ন গবেষণা প্রতিষ্ঠান ও বিজ্ঞানীদের মধ্যে তথ্য সহজে আদান-প্রদানের ব্যবস্থা তৈরি করা। ১৯৯০ সালে Tim Berners-Lee ও Robert Cailliau মিলে Web-এর ধারণাকে formal proposal আকারে উপস্থাপন করেন। Web-এর শুরুতে HTML ব্যবহার করা হয় hypertext document তৈরি করার জন্য, যাতে একটি document থেকে link-এর মাধ্যমে অন্য document-এ যাওয়া যায়।
\end{keypointbox}

\begin{boardbox}{HTML কে তৈরি করেন?}
\textbf{Tim Berners-Lee} HTML-এর মূল উদ্ভাবক হিসেবে পরিচিত। তিনি World Wide Web-এর জন্য তিনটি মৌলিক ধারণা একসাথে ব্যবহার করেন: \textbf{HTML} document structure-এর জন্য, \textbf{HTTP} browser-server communication-এর জন্য এবং \textbf{URL} web resource address করার জন্য। তাই HTML-এর history বলতে Tim Berners-Lee, CERN, WWW, HTTP ও URL--এই বিষয়গুলো একসাথে মনে রাখতে হবে।
\end{boardbox}

\subsection{HTML ব্যবহারের সুবিধা}

HTML শেখা সহজ এবং web page তৈরির সবচেয়ে মৌলিক language। HSC ICT practical-এ HTML গুরুত্বপূর্ণ, কারণ অল্প কিছু tag শিখেই heading, paragraph, image, hyperlink, list, table ও form তৈরি করা যায়।

\begin{keypointbox}{HTML-এর প্রধান সুবিধা}
\begin{itemize}
\item HTML ব্যবহার করা ও বোঝা তুলনামূলক সহজ।
\item HTML plain text ভিত্তিক, তাই সাধারণ editor দিয়ে code লেখা ও edit করা যায়।
\item প্রায় সব web browser HTML support করে।
\item সাধারণ text editor, যেমন Notepad বা VS Code দিয়েও HTML file লেখা যায়।
\item HTML file \texttt{.html} বা \texttt{.htm} extension দিয়ে save করা যায়।
\item HTML দিয়ে static webpage দ্রুত তৈরি করা যায়।
\item HTML page সাধারণত দ্রুত load হয়, কারণ basic HTML file lightweight।
\item HTML tag ব্যবহার করে content-এর structure পরিষ্কার করা যায়।
\item HTML tag case-sensitive নয়; \texttt{<P>} ও \texttt{<p>} browser সাধারণত একইভাবে বুঝতে পারে।
\item Search engine সাধারণত HTML structure সহজে পড়তে পারে।
\item CSS ও JavaScript-এর সাথে HTML ব্যবহার করে সুন্দর ও interactive webpage তৈরি করা যায়।
\end{itemize}
\end{keypointbox}

\begin{boardbox}{Board focus}
HTML-এর সুবিধা লিখতে বললে সহজ ভাষায় লিখবে: HTML সহজ, সব browser support করে, text editor দিয়ে লেখা যায়, webpage structure তৈরি করে এবং CSS/JavaScript-এর সাথে ব্যবহার করে পূর্ণাঙ্গ website তৈরি করা যায়।
\end{boardbox}

\subsection{HTML ট্যাগের তালিকা ও উদাহরণ}

নিচে বেশিরভাগ সাধারণ HTML ট্যাগের তালিকা, প্রতিটির সংক্ষিপ্ত কাজ, একেকটির একটি নমুনা HTML কোড এবং ব্রাউজারে কীভাবে প্রদর্শিত হবে (output) তা দেখানো হলো।

\begin{keypointbox}{প্রধান HTML ট্যাগ (সংক্ষেপ)}
\begin{itemize}
\item \texttt{<html>, <head>, <body>} : ডকুমেন্টের মূল কঙ্কাল
\item \texttt{<h1>.. <h6>} : শিরোনাম
\item \texttt{<p>} : প্যারাগ্রাফ
\item \texttt{<a>} : হাইপারলিংক
\item \texttt{<img>} : ছবি বসানো
\item \texttt{<ul>, <ol>, <li>} : তালিকা
\item \texttt{<table>, <tr>, <td>, <th>} : টেবিল
\item \texttt{<form>, <input>, <button>, <label>, <select>, <option>} : ফর্ম উপাদান
\item \texttt{<div>, <span>} : কন্টেইনার, inline/block
\item \texttt{<strong>, <em>} : গুরুত্ব ও emphasis
\item \texttt{<br>, <hr>} : লাইনে বিরতি, অনুভূমিক রেখা
\item \texttt{<pre>, <code>} : কোড/প্রি-ফরম্যাটেড টেক্সট
\item \texttt{<header>, <nav>, <main>, <section>, <article>, <aside>, <footer>} : semantic layout tags
\end{itemize}
\end{keypointbox}

\begin{examplebox}{ট্যাগগুলোর নমুনা ও output}
\textbf{1. শিরোনাম (headings)}\\
\begin{verbatim}
<h1>বড় শিরোনাম</h1>
<h2>ছোট শিরোনাম</h2>
\end{verbatim}
Output: বড় শিরোনাম (বড় ফন্ট), নিচে ছোট শিরোনাম (ছোট ফন্ট)।

\vspace{6pt}
\textbf{2. প্যারাগ্রাফ}\\
\begin{verbatim}
<p>এইটা একটি প্যারাগ্রাফ। HTML-এ প্যারাগ্রাফ আলাদা লাইনে দেখায়।</p>
\end{verbatim}
Output: একটি বাক্য লাগানো পৃষ্ঠা পরিবেশে আলাদা প্যারাগ্রাফ হিসেবে দেখা যাবে।

\vspace{6pt}
\textbf{3. লিংক}\\
\begin{verbatim}
<a href="https://www.example.com">Example সাইট</a>
\end{verbatim}
Output: ক্লিকযোগ্য "Example সাইট" লিংক, ক্লিক করলে নির্দিষ্ট URL-এ যাবে।

\vspace{6pt}
\textbf{4. ছবি}\\
\begin{verbatim}
<img src="images/logo.png" alt="Logo" width="200">
\end{verbatim}
Output: নির্দিষ্ট পথের ছবি প্রদর্শন (alt পাঠ্য তখনই দেখায় যদি ছবি লোড না হয়)।

\vspace{6pt}
\textbf{5. তালিকা (unordered এবং ordered)}\\
\begin{verbatim}
<ul>
	<li>প্রথম আইটেম</li>
	<li>দ্বিতীয় আইটেম</li>
</ul>
<ol>
	<li>প্রথম</li>
	<li>দ্বিতীয়</li>
</ol>
\end{verbatim}
Output: বুলেট তালিকা ও নম্বর তালিকা আলাদাভাবে প্রদর্শিত হবে।

\vspace{6pt}
\textbf{6. টেবিল}\\
\begin{verbatim}
<table border="1">
	<tr><th>নাম</th><th>বয়স</th></tr>
	<tr><td>আজ</td><td>16</td></tr>
</table>
\end{verbatim}
Output: সারি ও কলাম সহ একটি টেবিল দেখাবে, প্রথম সারিটি হেডার হিসেবে।

\vspace{6pt}
\textbf{7. ফর্ম (input)}\\
\begin{verbatim}
<form action="/submit">
	<label>Name: <input type="text" name="name"></label>
	<button type="submit">Send</button>
</form>
\end{verbatim}
Output: একটি টেক্সট ইনপুট বক্স এবং সাবমিট বাটন প্রদর্শিত হবে; সাবমিট করলে action-এ পাঠাবে।

\vspace{6pt}
\textbf{8. div এবং span}\\
\begin{verbatim}
<div style="background:#eef;padding:10px;">Block level container</div>
<span style="color:red;">Inline text</span>
\end{verbatim}
Output: \texttt{div} নতুন ব্লকে কন্টেন্ট রাখে; \texttt{span} inline text style দেয়।

\vspace{6pt}
\textbf{9. emphasis এবং strong}\\
\begin{verbatim}
<p>এটা <strong>গুরুত্বপূর্ণ</strong> এবং এটা <em>জোর দিয়ে</em> বলা।</p>
\end{verbatim}
Output: \texttt{strong} সাধারণত bold, \texttt{em} italic রেন্ডার করে।

\vspace{6pt}
\textbf{10. break এবং horizontal rule}\\
\begin{verbatim}
Line 1<br>Line 2
<hr>
\end{verbatim}
Output: লাইনে বিরতি এবং অনুভূমিক লাইনের মতো বিভাজক দেখা যাবে।

\vspace{6pt}
\textbf{11. pre এবং code}\\
\begin{verbatim}
<pre>
function greet() {
	console.log('Hello');
}
</pre>
<p>Inline code: <code>const x = 5;</code></p>
\end{verbatim}
Output: \texttt{pre} ব্লক যেখানে স্পেস ও লাইনের ফরম্যাট রাখা হয়; \texttt{code} inline monospace টেক্সট দেখায়।

\vspace{6pt}
\textbf{12. semantic layout tags (সংক্ষিপ্ত)}\\
\begin{verbatim}
<header>Header content</header>
<nav>Navigation links</nav>
<main>Main article</main>
<aside>Sidebar</aside>
<footer>Footer info</footer>
\end{verbatim}
Output: ব্রাউজারে এগুলো সাধারণly block-level হিসেবে প্রদর্শিত হবে; screen reader ও SEO-এর জন্য semantic সুবিধা দেয়।
\end{examplebox}
