\subsection{Final Tag Reference}

Chapter 4-এর HTML practical অংশে tag মুখস্থ করার চেয়ে tag-এর কাজ, syntax, output এবং problem-solving use বুঝে শেখা বেশি গুরুত্বপূর্ণ। নিচের tableটি final revision হিসেবে ব্যবহার করা যাবে।

\begin{center}
\begin{tabular}{|p{0.14\textwidth}|p{0.24\textwidth}|p{0.23\textwidth}|p{0.25\textwidth}|}
\hline
\textbf{Tag/Attribute} & \textbf{Definition/কাজ} & \textbf{Common syntax} & \textbf{কী কী করা যায়} \\
\hline
\texttt{<html>} & সম্পূর্ণ HTML document ধরে & \texttt{<html>...</html>} & page-এর root structure তৈরি \\

\hline

\hline

\hline

\subsection{Problem Solution Format}

Board practical বা CQ-তে output দেখে code লিখতে হলে নিচের format follow করবে।

\begin{practicebox}
\textbf{Problem:} নিচের output তৈরি করো--একটি college webpage যেখানে heading, menu link, image, subject list এবং result table থাকবে।

\textbf{Step 1: Output analyze করো}\\
Heading আছে, menu link আছে, image আছে, list আছে, table আছে।

\textbf{Step 2: Tag select করো}\\
\texttt{<h1>} heading-এর জন্য, \texttt{<a>} menu link-এর জন্য, \texttt{<img>} image-এর জন্য, \texttt{<ul>} ও \texttt{<li>} list-এর জন্য, \texttt{<table>}, \texttt{<tr>}, \texttt{<th>}, \texttt{<td>} table-এর জন্য।

\textbf{Step 3: Skeleton লেখো}\\
\begin{examplebox}{Code}
\begin{verbatim}
<!DOCTYPE html>
<html>
<head>
  <title>College Page</title>
</head>
<body>

</body>
</html>
\end{verbatim}
\end{examplebox}

\textbf{Step 4: Section অনুযায়ী code বসাও}\\
প্রথমে header, তারপর menu, তারপর image/list/table, শেষে footer লিখবে।

\textbf{Step 5: Output verify করো}\\
Browser-এ দেখে row/column, image path, link address, spelling ও closing tag মিলিয়ে নেবে।
\end{practicebox}

\subsection{Find the Mistake}

\begin{practicebox}
\textbf{ভুল code:}
\begin{examplebox}{Code}
\begin{verbatim}
<table border="1">
  <td>Name</td>
  <td>Roll</td>
</table>

<image src="photo.jpg">
<a src="index.html">Home</a>
\end{verbatim}
\end{examplebox}

\textbf{সমাধান:}
\begin{itemize}
\item \texttt{<td>} সরাসরি \texttt{<table>} tag-এর ভিতরে লেখা যাবে না; \texttt{<tr>} দরকার।
\item image দেখানোর tag \texttt{<image>} নয়, সঠিক tag হলো \texttt{<img>}.
\item hyperlink-এর জন্য \texttt{src} নয়, \texttt{href} attribute ব্যবহার করতে হয়।
\end{itemize}

\textbf{সঠিক code:}
\begin{examplebox}{Code}
\begin{verbatim}
<table border="1">
  <tr>
    <td>Name</td>
    <td>Roll</td>
  </tr>
</table>

<img src="photo.jpg" alt="Photo">
<a href="index.html">Home</a>
\end{verbatim}
\end{examplebox}
\end{practicebox}

\subsection{Source PDF মিলিয়ে CQ Pattern}

২০২৩-২০২৫ board-style CQ-তে Chapter 4 থেকে বারবার কয়েক ধরনের practical problem এসেছে। এগুলো আলাদা করে practice করলে output দেখে code লেখা সহজ হয়।

\begin{center}
\begin{tabular}{|p{0.25\textwidth}|p{0.34\textwidth}|p{0.25\textwidth}|}
\hline
\textbf{Pattern} & \textbf{যা করতে হয়} & \textbf{প্রধান tag/attribute} \\
\hline
Nested list & category-এর ভিতরে subcategory দেখানো & \texttt{<ul>}, \texttt{<ol>}, \texttt{<li>}, \texttt{type}, \texttt{start} \\
\hline
List থেকে table & list item গুলো row/column আকারে দেখানো & \texttt{<table>}, \texttt{<tr>}, \texttt{<td>}, \texttt{rowspan} \\
\hline
Result/marks table & title, heading, marks, merged cell & \texttt{colspan}, \texttt{rowspan}, \texttt{cellpadding}, \texttt{cellspacing} \\
\hline
Image link & image-এ click করলে website/page open & \texttt{<a href="">}, \texttt{<img src="">} \\
\hline
Full webpage & header, menu, image, list, table, footer & \texttt{<div>}, \texttt{<a>}, \texttt{<img>}, \texttt{<table>} \\
\hline
Publishing issue & website নির্দিষ্ট সময় পর দেখা যাচ্ছে না কেন & domain expiry, hosting expiry, DNS/server problem \\
\hline
\end{tabular}
\end{center}

\begin{practicebox}
\textbf{CQ Pattern Practice: List থেকে Table}\\
Output-এ River cell তিনটি row জুড়ে থাকবে এবং পাশে Padma, Jamuna, Meghna থাকবে। Flower cell তিনটি row জুড়ে থাকবে এবং পাশে Adenium, Rose, Lily থাকবে।

\textbf{Code:}
\begin{examplebox}{Code}
\begin{verbatim}
<!DOCTYPE html>
<html>
<body>
  <table border="1" cellspacing="0" cellpadding="5">
    <tr>
      <td rowspan="3" align="center">River</td>
      <td>Padma</td>
    </tr>
    <tr>
      <td>Jamuna</td>
    </tr>
    <tr>
      <td>Meghna</td>
    </tr>
    <tr>
      <td rowspan="3" align="center">Flower</td>
      <td>Adenium</td>
    </tr>
    <tr>
      <td>Rose</td>
    </tr>
    <tr>
      <td>Lily</td>
    </tr>
  </table>
</body>
</html>
\end{verbatim}
\end{examplebox}

\textbf{কেন \texttt{rowspan="3"}?}\\
River-এর পাশে তিনটি river name আছে, তাই River cell তিনটি row জুড়ে থাকবে। Flower-এর ক্ষেত্রেও একই কারণে \texttt{rowspan="3"} ব্যবহার করা হয়েছে।
\end{practicebox}

\begin{recapbox}
\begin{itemize}
\item HTML structure দেয়, CSS style দেয়, JavaScript behavior যোগ করে।
\item \texttt{href} link-এর গন্তব্য, \texttt{src} image/file path, \texttt{alt} বিকল্প text।
\item Table: \texttt{table} $\rightarrow$ \texttt{tr} $\rightarrow$ \texttt{th}/\texttt{td}।
\item Form: input গ্রহণ করে; GET search/filter-এ, POST sensitive submission-এ বেশি উপযোগী।
\item Publish করার আগে link, image, form, mobile view ও browser compatibility test করতে হয়।
\end{itemize}
\end{recapbox}
