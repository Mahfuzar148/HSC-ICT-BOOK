\subsection{Layout কী?}

\begin{definitionbox}{Webpage Layout}
ওয়েবপেজে header, navigation menu, main content, image, table, sidebar ও footer কোথায় থাকবে, সেই পরিকল্পিত বিন্যাসকে webpage layout বলে।
\end{definitionbox}

ভালো layout page-কে পরিষ্কার, পড়ার উপযোগী এবং ব্যবহারকারী-বান্ধব করে। HSC practical-এ full webpage design করার আগে layout বুঝে নিলে code লেখা সহজ হয়।

\begin{itemize}
\item code লেখার আগে page plan পরিষ্কার হয়।
\item কোন content কোথায় থাকবে তা বোঝা যায়।
\item header, menu, content ও footer আলাদা করা সহজ হয়।
\item figure দেখে HTML code লেখার সময় ভুল কম হয়।
\item বড় project-এ designer ও developer একই plan follow করতে পারে।

\subsection{Responsive Design কী?}

\begin{definitionbox}{Responsive Design}
যে design-এ webpage-এর layout screen size অনুযায়ী পরিবর্তিত হয়ে desktop, tablet ও mobile-এ সুন্দরভাবে দেখা যায়, তাকে responsive design বলে।
\end{definitionbox}

Responsive design-এ সাধারণত flexible width, readable font, properly sized image, mobile-friendly menu এবং viewport meta tag ব্যবহার করা হয়।

\begin{examplebox}{Code}
\begin{verbatim}
<meta name="viewport" content="width=device-width, initial-scale=1.0">
\end{verbatim}
\end{examplebox}

\subsection{ভালো Web Layout-এর Design Principle}

\begin{keypointbox}{Design principle}
\begin{itemize}
\item \textbf{Contrast:} গুরুত্বপূর্ণ অংশ আলাদা করে দেখানো।
\item \textbf{Alignment:} text ও element সুশৃঙ্খল রাখা।
\item \textbf{Repetition:} একই style বারবার ব্যবহার করে consistency রাখা।
\item \textbf{Proximity:} সম্পর্কিত content কাছাকাছি রাখা।
\item \textbf{Whitespace:} content-এর মাঝে পর্যাপ্ত ফাঁকা জায়গা রাখা।
\item \textbf{Readability:} text যেন সহজে পড়া যায়।
\end{itemize}
\end{keypointbox}

\subsection{Problem Solution}

\begin{practicebox}
\textbf{Problem:} একটি college webpage-এর layout plan তৈরি করো যেখানে header, menu, notice, subject list, result table ও footer থাকবে।

\textbf{Solution plan:}
\begin{enumerate}
\item Header অংশে \texttt{<div>} এর ভিতরে \texttt{<h1>} দিয়ে college name লিখবে।
\item Menu অংশে \texttt{<a>} দিয়ে Home, Notice, Result, Contact link লিখবে।
\item Main content অংশে notice paragraph লিখবে।
\item Subject list-এর জন্য \texttt{<ul>} ও \texttt{<li>} ব্যবহার করবে।
\item Result table-এর জন্য \texttt{<table>}, \texttt{<tr>}, \texttt{<th>}, \texttt{<td>} ব্যবহার করবে।
\item Footer অংশে copyright বা email লিখবে।
\end{enumerate}
\end{practicebox}

\begin{boardbox}{Board focus}
Webpage design practical-এ code শুরু করার আগে layout section identify করা খুব জরুরি। Header, menu, content, table/list/image ও footer আলাদা করলে code clean হয় এবং output figure-এর সাথে মেলানো সহজ হয়।
\end{boardbox}

\begin{mistakebox}{সাধারণ ভুল}
\begin{itemize}
\item layout plan না করে সরাসরি code লেখা।
\item header, menu, content ও footer আলাদা না করা।
\item সব content এক \texttt{<div>} tag-এর ভিতরে অগোছালোভাবে লেখা।
\item mobile view চিন্তা না করা।
\item image width বেশি দিয়ে page overflow করা।
\item menu link-এ \texttt{href} না দেওয়া।
\end{itemize}
\end{mistakebox}
