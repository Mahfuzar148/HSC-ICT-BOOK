# Chapter 4 — ওয়েব ডিজাইন পরিচিতি ও HTML

This file lists the finalized, ordered topics for Chapter 4 (Web design & HTML), arranged for HSC ICT textbook-style teaching.

## Final Topic Order

1. ওয়েব ডিজাইন পরিচিতি
2. ওয়েব পেজ, ওয়েবসাইট, হোম পেজ, ওয়েব সার্ভার
3. WWW, Web Browser, Search Engine, Web Portal
4. IP Address, Domain Name, URL, HTTP, HTTPS, FTP, DNS
5. Static ও Dynamic Website
6. Local ও Remote Website
7. Website Structure: Linear, Tree, Web-linked, Hybrid
8. HTML-এর ধারণা, tag, element, attribute
9. Basic HTML Structure: `html`, `head`, `title`, `body`
10. Text formatting tags (headings, paragraph, break, bold, italic, etc.)
11. Lists (`ol`, `ul`, `dl`)
12. Images (`<img>`, `src`, `alt`, sizing`, `figure`/`figcaption`)
13. Hyperlinks (`<a>`, `href`, `target`, `rel`)
14. Tables (`<table>`, `tr`, `th`, `td`, `caption`, `border`, `cellspacing`, `cellpadding`)
15. Rowspan ও Colspan (`rowspan`, `colspan`, table cell merge)
16. Table-এর ভিতরে List, Image, Hyperlink ও Nested Table
17. Division, Span ও Style Attribute (`<div>`, `<span>`, inline CSS basic)
18. Full Webpage Designing Practical (Header, Menu, Image, Table, List ও Link)
19. Forms (`<form>`, `input`, `textarea`, `select`, `button`, GET/POST`)
20. Web page design & development (wireframe, layout, prototype)
21. Website publishing (domain, hosting, upload, DNS)
22. Board-style HTML code practice & exercises
