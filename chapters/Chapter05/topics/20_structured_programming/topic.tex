\section{Structured Programming}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী structured programming-এর control structure ও সুবিধা ব্যাখ্যা করতে পারবে।
\end{learningbox}

\begin{definitionbox}{Structured programming}
যে পদ্ধতিতে program-কে ছোট module-এ ভাগ করে sequence, selection ও iteration control structure দিয়ে সাজানো হয়, তাকে structured programming বলা হয়।
\end{definitionbox}

\begin{keypointbox}{তিন control structure}
\begin{itemize}
\item Sequence: instruction ধারাবাহিকভাবে চলে।
\item Selection: condition অনুযায়ী branch নেয়।
\item Loop/Iteration: একই কাজ বারবার করে।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{C language}
C language structured programming support করে; function ব্যবহার করে বড় program ছোট অংশে ভাগ করা যায়।
\end{examplebox}
