\section{অনুবাদক (Translator) প্রোগ্রাম}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী translator program-এর কাজ এবং compiler, interpreter, assembler-এর পার্থক্য বুঝতে পারবে।
\end{learningbox}

\begin{definitionbox}{Translator program}
যে system software source program-কে machine language বা object program-এ রূপান্তর করে, তাকে translator program বলা হয়।
\end{definitionbox}

\begin{boardbox}{বোর্ড ফোকাস}
Translator-এর প্রশ্নে source program, object program এবং machine language শব্দগুলো ব্যবহার করো।
\end{boardbox}
