% ১৪.৮ Expression
\section{Expression}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী expression, precedence ও associativity-এর ধারণা ব্যাখ্যা করতে পারবে।
\end{learningbox}

\begin{definitionbox}{Expression}
Constant, variable, operator ও function call-এর সমন্বয়ে যে valid statement অংশ একটি value তৈরি করে, তাকে expression বলা হয়।
\end{definitionbox}

\begin{examplebox}{Expression}
\texttt{sum = a + b * c;} expression-এ multiplication আগে হবে, তারপর addition। কারণ \texttt{*}-এর precedence \texttt{+}-এর চেয়ে বেশি।
\end{examplebox}

\begin{keypointbox}{ধরন}
\begin{itemize}
\item Arithmetic expression: \texttt{a+b}
\item Relational expression: \texttt{a>b}
\item Logical expression: \texttt{a>b \&\& b>c}
\item Assignment expression: \texttt{x=10}
\end{itemize}
\end{keypointbox}
