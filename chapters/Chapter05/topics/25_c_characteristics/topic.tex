% ১৪.১ C ভাষার বৈশিষ্ট্য
\section{C ভাষার বৈশিষ্ট্য}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী C ভাষার প্রধান বৈশিষ্ট্য ও সুবিধা লিখতে পারবে।
\end{learningbox}

\begin{keypointbox}{C ভাষার বৈশিষ্ট্য}
\begin{itemize}
\item Mid-level: high-level ও low-level দুই বৈশিষ্ট্য আছে।
\item Structured: function ও block দিয়ে program সাজানো যায়।
\item Portable: সামান্য পরিবর্তনে ভিন্ন machine-এ চালানো যায়।
\item Fast: compiled language হওয়ায় execution দ্রুত।
\item Rich library: standard library function আছে।
\item Case sensitive: \texttt{Sum} ও \texttt{sum} আলাদা।
\end{itemize}
\end{keypointbox}

\begin{mistakebox}{সাধারণ ভুল}
C ভাষায় uppercase ও lowercase আলাদা। \texttt{Printf} লিখলে \texttt{printf}-এর মতো কাজ করবে না।
\end{mistakebox}
