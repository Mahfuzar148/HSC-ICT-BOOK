\section{চতুর্থ প্রজন্মের ভাষা (4GL)}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী 4GL-এর ধারণা, ব্যবহার ও 3GL-এর সাথে পার্থক্য লিখতে পারবে।
\end{learningbox}

\begin{definitionbox}{4GL}
যে very high-level language-এ কম code লিখে query, report, database বা business application তৈরি করা যায়, তাকে fourth generation language বা 4GL বলা হয়।
\end{definitionbox}

\begin{center}
\begin{tabular}{|p{0.25\textwidth}|p{0.30\textwidth}|p{0.30\textwidth}|}
\hline
\textbf{ভিত্তি} & \textbf{3GL} & \textbf{4GL} \\
\hline
Code & বেশি detailed code & কম code, বেশি declarative \\
\hline
ব্যবহার & general program & database/query/report \\
\hline
উদাহরণ & C, Java & SQL, Oracle tools \\
\hline
\end{tabular}
\end{center}

\begin{examplebox}{SQL example}
\texttt{SELECT name, marks FROM students;} — এই ধরনের statement দিয়ে database থেকে প্রয়োজনীয় data বের করা যায়।
\end{examplebox}
