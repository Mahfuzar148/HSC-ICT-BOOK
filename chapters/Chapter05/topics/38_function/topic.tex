% ১৪.১৪ Function
\section{Function}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী function declaration, definition, calling, parameter ও return value বুঝতে পারবে।
\end{learningbox}

\begin{definitionbox}{Function}
নির্দিষ্ট কাজ সম্পাদনের জন্য program-এর named reusable block-কে function বলা হয়।
\end{definitionbox}

\begin{verbatim}
int add(int a, int b)
{
    return a + b;
}
\end{verbatim}

\begin{keypointbox}{Function-এর অংশ}
\begin{itemize}
\item Function declaration/prototype
\item Function definition
\item Function call
\item Parameter/argument
\item Return value
\end{itemize}
\end{keypointbox}

\begin{examplebox}{সুবিধা}
একই calculation program-এর একাধিক জায়গায় দরকার হলে function ব্যবহার করলে code reuse হয় এবং debugging সহজ হয়।
\end{examplebox}
