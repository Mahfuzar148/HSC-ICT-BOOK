\section{Keyword}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী keyword ও identifier-এর পার্থক্য লিখতে পারবে।
\end{learningbox}

\begin{definitionbox}{Keyword}
C language-এ পূর্বনির্ধারিত অর্থসহ সংরক্ষিত শব্দ, যা variable বা function name হিসেবে ব্যবহার করা যায় না, তাকে keyword বলা হয়।
\end{definitionbox}

\begin{keypointbox}{Common keyword}
\texttt{int}, \texttt{float}, \texttt{char}, \texttt{if}, \texttt{else}, \texttt{for}, \texttt{while}, \texttt{return}, \texttt{break}, \texttt{continue}, \texttt{switch}, \texttt{case}, \texttt{void}
\end{keypointbox}

\begin{examplebox}{Identifier বনাম keyword}
\texttt{marks} identifier হতে পারে, কিন্তু \texttt{int} keyword হওয়ায় variable name হিসেবে ব্যবহার করা যাবে না।
\end{examplebox}
