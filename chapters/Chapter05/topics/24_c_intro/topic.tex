\section{C প্রোগ্রামিং — প্রাথমিক ধারণা}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী C language-এর পরিচিতি, উদ্ভাবক ও ব্যবহার লিখতে পারবে।
\end{learningbox}

\begin{definitionbox}{C language}
C হলো general-purpose, structured এবং mid-level programming language, যা 1970-এর দশকে Dennis Ritchie Bell Labs-এ তৈরি করেন।
\end{definitionbox}

\begin{keypointbox}{কেন C শেখা হয়}
\begin{itemize}
\item syntax তুলনামূলক সংক্ষিপ্ত ও শক্তিশালী।
\item system programming ও general programming দুটোতেই ব্যবহারযোগ্য।
\item programming logic শেখার জন্য কার্যকর।
\item অনেক আধুনিক ভাষার syntax C দ্বারা প্রভাবিত।
\end{itemize}
\end{keypointbox}

\begin{boardbox}{বোর্ড ফোকাস}
C ভাষার পরিচিতিতে Dennis Ritchie, structured, mid-level, portable ও general-purpose শব্দগুলো গুরুত্বপূর্ণ।
\end{boardbox}
