\section{প্রোগ্রামিং ভাষার ধারণা}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী programming language-এর প্রয়োজন, syntax এবং human-readable code থেকে machine code-এ যাওয়ার ধারণা বুঝবে।
\end{learningbox}

\begin{definitionbox}{প্রোগ্রামিং ভাষা}
নির্দিষ্ট নিয়ম, শব্দ, symbol ও syntax ব্যবহার করে কম্পিউটারের জন্য program লেখার ভাষাকে programming language বলা হয়।
\end{definitionbox}

\begin{center}
\fbox{\parbox{0.9\textwidth}{
\centering
\textbf{Translation model placeholder}\\[4pt]
Human problem $\rightarrow$ Programming language code $\rightarrow$ Translator $\rightarrow$ Machine code $\rightarrow$ Execution
}}
\end{center}

\begin{keypointbox}{প্রয়োজনীয়তা}
\begin{itemize}
\item জটিল সমস্যা ধাপে ধাপে computer-কে বোঝানো যায়।
\item একই logic বারবার ব্যবহারযোগ্য program হিসেবে লেখা যায়।
\item business, science, education, mobile app ও web application তৈরি করা যায়।
\end{itemize}
\end{keypointbox}

\begin{mistakebox}{সাধারণ ভুল}
কম্পিউটার সরাসরি বাংলা বা ইংরেজি বোঝে না; translator-এর মাধ্যমে high-level code machine code-এ রূপান্তরিত হয়।
\end{mistakebox}
