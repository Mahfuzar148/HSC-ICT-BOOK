% ১৬. অনুশীলনী
\section{অনুশীলনী}

\begin{learningbox}
এই অনুশীলন শেষে শিক্ষার্থী programming language theory, algorithm, flowchart এবং C code নিয়ে board-style প্রশ্ন সমাধান করতে পারবে।
\end{learningbox}

\begin{practicebox}
\textbf{MCQ}
\begin{enumerate}
\item C language-এর উদ্ভাবক কে?\\
ক. Charles Babbage \quad খ. Dennis Ritchie \quad গ. George Boole \quad ঘ. Tim Berners-Lee
\item Assembly language translate করে কোনটি?\\
ক. Compiler \quad খ. Interpreter \quad গ. Assembler \quad ঘ. Linker
\item \texttt{scanf}-এ variable address দিতে কোন operator ব্যবহৃত হয়?\\
ক. \texttt{*} \quad খ. \texttt{\&} \quad গ. \texttt{\%} \quad ঘ. \texttt{!}
\item C array-এর first index কত?\\
ক. 0 \quad খ. 1 \quad গ. -1 \quad ঘ. 2
\end{enumerate}
\end{practicebox}

\begin{practicebox}
\textbf{সংক্ষিপ্ত প্রশ্ন}
\begin{enumerate}
\item Compiler ও interpreter-এর পার্থক্য লেখ।
\item Algorithm-এর বৈশিষ্ট্য লেখ।
\item Flowchart-এর decision symbol কোনটি?
\item Variable naming rule লেখ।
\item \texttt{for} ও \texttt{while} loop-এর ব্যবহার ব্যাখ্যা কর।
\end{enumerate}
\end{practicebox}

\begin{practicebox}
\textbf{সৃজনশীল অনুশীলন}\\
একজন শিক্ষার্থী তিনটি সংখ্যা input নিয়ে সবচেয়ে বড় সংখ্যা বের করার program লিখতে চায়। সে প্রথমে algorithm লিখে, পরে flowchart এঁকে C code লিখল।
\begin{enumerate}
\item ক. Algorithm কী?
\item খ. Flowchart কেন coding-এর আগে সহায়ক?
\item গ. উদ্দীপকের problem-এর algorithm লেখ।
\item ঘ. একই problem-এর জন্য \texttt{if...else} ব্যবহার করে C code লেখ।
\end{enumerate}
\end{practicebox}

\begin{recapbox}
\begin{itemize}
\item Program হলো instruction set; programming language দিয়ে program লেখা হয়।
\item Compiler পুরো program, interpreter line by line, assembler assembly code translate করে।
\item Algorithm আগে logic পরিষ্কার করে, flowchart logic চিত্রে দেখায়।
\item C program-এ \texttt{main()}, declaration, input, processing, output ও return থাকে।
\item Condition, loop, array, function Chapter 5-এর practical core।
\end{itemize}
\end{recapbox}
