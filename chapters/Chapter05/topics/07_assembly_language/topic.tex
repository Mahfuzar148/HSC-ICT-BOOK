\section{Assembly Language}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী assembly language, mnemonic code ও assembler-এর ভূমিকা ব্যাখ্যা করতে পারবে।
\end{learningbox}

\begin{definitionbox}{Assembly language}
যে low-level language-এ binary instruction-এর বদলে mnemonic code যেমন \texttt{ADD}, \texttt{MOV}, \texttt{SUB} ব্যবহার করা হয়, তাকে assembly language বলা হয়।
\end{definitionbox}
