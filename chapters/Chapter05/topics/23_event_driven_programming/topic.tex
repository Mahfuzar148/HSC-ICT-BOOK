\section{Event Driven Programming}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী event, event handler এবং GUI application-এর event-driven flow ব্যাখ্যা করতে পারবে।
\end{learningbox}

\begin{definitionbox}{Event-driven programming}
যে programming model-এ mouse click, key press, form submit, timer ইত্যাদি event ঘটলে নির্দিষ্ট handler/function চালু হয়, তাকে event-driven programming বলা হয়।
\end{definitionbox}

\begin{center}
\fbox{\parbox{0.88\textwidth}{
\centering
\textbf{Event flow placeholder}\\[4pt]
User click $\rightarrow$ Event generated $\rightarrow$ Event handler executes $\rightarrow$ Output changes
}}
\end{center}

\begin{examplebox}{উদাহরণ}
Button-এ click করলে calculator app যোগফল দেখায়। এখানে click হলো event এবং যোগফল দেখানোর code হলো event handler।
\end{examplebox}
