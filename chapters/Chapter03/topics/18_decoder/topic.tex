\section{Decoder}

\subsection{সংজ্ঞা}
\begin{itemize}
\item Decoder: n-bit input থেকে 2\textsuperscript{n} output
\item 2-to-4, 3-to-8 decoder উদাহরণ
\end{itemize}

\subsection{ব্যবহার}
\begin{itemize}
\item Memory address decoding, display selection
\end{itemize}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী পারবে:
\begin{itemize}
\item decoder-এর input-output relation লিখতে।
\item $n$ input থেকে $2^n$ output ধারণা ব্যাখ্যা করতে।
\item encoder ও decoder-এর পার্থক্য লিখতে।
\end{itemize}
\end{learningbox}

\begin{definitionbox}{Decoder}
যে combinational circuit binary input code গ্রহণ করে নির্দিষ্ট একটি output line সক্রিয় করে, তাকে decoder বলা হয়। সাধারণ decoder-এ $n$ input এবং $2^n$ output থাকে।
\end{definitionbox}
