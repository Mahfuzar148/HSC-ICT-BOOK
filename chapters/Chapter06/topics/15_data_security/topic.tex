% ১৫. Data Security
\section{Data Security}
ডেটা নিরাপত্তা হলো ডেটাকে অননুমোদিত প্রবেশ, পরিবর্তন, নষ্ট হওয়া, চুরি, প্রকাশ বা হারিয়ে যাওয়া থেকে রক্ষা করার ব্যবস্থা। ডেটাবেজ যত বড় ও গুরুত্বপূর্ণ হয়, নিরাপত্তার প্রয়োজনও তত বেশি হয়।

\subsection{Security কী ও প্রয়োজনীয়তা}
\begin{definitionbox}{Data Security}
ডেটার গোপনীয়তা, অখণ্ডতা এবং প্রাপ্যতা রক্ষা করার জন্য নেওয়া প্রযুক্তিগত ও প্রশাসনিক ব্যবস্থাকে Data Security বলে।
\end{definitionbox}

\subsection{Security পদ্ধতি}
\begin{keypointbox}{CIA Triad}
\begin{itemize}
    \item Confidentiality: অনুমোদিত ব্যক্তি ছাড়া কেউ ডেটা দেখতে পারবে না।
    \item Integrity: ডেটা সঠিক ও অপরিবর্তিত থাকবে।
    \item Availability: প্রয়োজনের সময় অনুমোদিত ব্যবহারকারী ডেটা পাবে।
\end{itemize}
\end{keypointbox}
