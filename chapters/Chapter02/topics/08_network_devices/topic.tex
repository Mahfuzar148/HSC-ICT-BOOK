\section{নেটওয়ার্ক ডিভাইস}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী পারবে:
\begin{itemize}
\item নেটওয়ার্ক ডিভাইস কী তা ব্যাখ্যা করতে।
\item NIC, Modem, Hub, Switch, Router, Repeater, Bridge, Gateway, Access Point ও Firewall-এর কাজ বুঝতে।
\item Hub, Switch ও Router-এর পার্থক্য লিখতে।
\item কোন device কোন network layer-এ কাজ করে তা ধারণা নিতে।
\item বাস্তব network scenario থেকে উপযুক্ত network device নির্বাচন করতে।
\item বোর্ড পরীক্ষার জন্য সংক্ষিপ্ত, পার্থক্যভিত্তিক ও সৃজনশীল উত্তর লিখতে।
\end{itemize}
\end{learningbox}

\subsection{বাস্তব জীবনের শুরু}

একটি computer lab কল্পনা করো। সেখানে অনেকগুলো computer আছে। সব computer-কে একসাথে network-এ যুক্ত করতে একটি \textbf{switch} ব্যবহার করা হয়। আবার lab-এর network-কে internet-এর সাথে যুক্ত করতে \textbf{router} ও \textbf{modem} ব্যবহার করা হয়। যদি wireless internet দিতে হয়, তাহলে \textbf{access point} বা Wi-Fi router দরকার হয়।

অর্থাৎ শুধু computer থাকলেই network তৈরি হয় না। Computer, server, printer, internet line এবং wireless device-গুলোকে যুক্ত, নিয়ন্ত্রণ, signal বৃদ্ধি, data forward এবং network নিরাপদ রাখতে বিভিন্ন hardware device প্রয়োজন হয়। এগুলোই হলো \textbf{network devices}।

\begin{keypointbox}{মূল ধারণা}
Network devices বুঝতে চারটি প্রশ্ন মনে রাখো:
\begin{itemize}
\item device-টি network-এর ভিতরে কাজ করছে, নাকি ভিন্ন network connect করছে?
\item device-টি signal বাড়াচ্ছে, নাকি data ঠিক destination-এ পাঠাচ্ছে?
\item device-টি wired connection দিচ্ছে, নাকি wireless connection দিচ্ছে?
\item device-টি security বা filtering করছে কি না?
\end{itemize}
\end{keypointbox}

\subsection{নেটওয়ার্ক ডিভাইস}

\begin{definitionbox}{নেটওয়ার্ক ডিভাইস}
কম্পিউটার নেটওয়ার্কে device, computer, server, printer বা বিভিন্ন network-কে সংযুক্ত করা, data আদান-প্রদান নিয়ন্ত্রণ করা, signal পুনরুদ্ধার করা, route নির্ধারণ করা এবং security নিশ্চিত করার জন্য ব্যবহৃত hardware বা hardware-software সমন্বিত উপাদানকে network device বলা হয়।
\end{definitionbox}

Network device-কে networking hardware-ও বলা হয়। এগুলো ছাড়া LAN, MAN, WAN, internet connection বা wireless network কার্যকরভাবে পরিচালনা করা কঠিন।

\begin{examplebox}{সহজ উদাহরণ}
\begin{itemize}
\item Computer lab-এ অনেক PC যুক্ত করতে switch ব্যবহার করা হয়।
\item বাসার Wi-Fi network-এ router internet ভাগ করে দেয়।
\item ISP line থেকে digital data নিতে modem ব্যবহার করা হয়।
\item বড় cable distance-এ দুর্বল signal পুনরায় শক্তিশালী করতে repeater ব্যবহার করা হয়।
\item ভিন্ন protocol বা ভিন্ন network system connect করতে gateway ব্যবহার করা হয়।
\end{itemize}
\end{examplebox}

\subsection{Network Device কেন প্রয়োজন?}

একটি network-এ data শুধু cable দিয়ে যায় না; data কোথায় যাবে, কোন device-এর কাছে যাবে, কোন network-এ যাবে, signal দুর্বল হলে কী হবে, unauthorized access আটকাবে কীভাবে—এসব কাজ network device দ্বারা সম্পন্ন হয়।

\begin{keypointbox}{Network device-এর প্রয়োজনীয়তা}
\begin{itemize}
\item একাধিক computer বা device network-এ যুক্ত করে।
\item data packet সঠিক destination-এ পৌঁছাতে সাহায্য করে।
\item local network-কে internet বা অন্য network-এর সাথে যুক্ত করে।
\item দুর্বল signal regenerate বা amplify করতে পারে।
\item wireless device-কে network access দেয়।
\item network traffic নিয়ন্ত্রণ করে performance বাড়ায়।
\item unauthorized access ও malicious traffic filter করতে পারে।
\end{itemize}
\end{keypointbox}

\subsection{Network Interface Card বা NIC}

\begin{definitionbox}{NIC}
Network Interface Card বা NIC হলো এমন একটি hardware component যার মাধ্যমে computer বা device network-এর সাথে যুক্ত হয়।
\end{definitionbox}

NIC computer-কে network medium-এর সাথে connect করে। প্রতিটি NIC-এর সাধারণত একটি unique physical address থাকে, যাকে MAC address বলা হয়। Wired NIC Ethernet cable-এর মাধ্যমে network-এ যুক্ত হয়, আর wireless NIC Wi-Fi signal ব্যবহার করে।

\begin{keypointbox}{NIC-এর বৈশিষ্ট্য}
\begin{itemize}
\item computer বা device-কে network-এ connect করে।
\item wired বা wireless দুই ধরনের হতে পারে।
\item MAC address ব্যবহার করে device identification করা যায়।
\item LAN communication-এ গুরুত্বপূর্ণ ভূমিকা রাখে।
\item motherboard-এ built-in থাকতে পারে বা আলাদা card/adapter হিসেবে থাকতে পারে।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{বাস্তব উদাহরণ}
Desktop computer-এর Ethernet port বা laptop-এর Wi-Fi adapter হলো NIC-এর উদাহরণ।

\subsection{Modem}

\begin{definitionbox}{Modem}
Modem হলো এমন একটি network device যা analog signal-কে digital signal-এ এবং digital signal-কে analog signal-এ রূপান্তর করে data communication সম্ভব করে।
\end{definitionbox}

Modem শব্দটি এসেছে Modulator এবং Demodulator থেকে। Internet connection-এ modem ISP-এর signal গ্রহণ করে user-এর device বা router-এর জন্য উপযোগী data signal তৈরি করে।

\begin{keypointbox}{Modem-এর কাজ}
\begin{itemize}
\item signal modulation ও demodulation করে।
\item ISP line এবং local network/device-এর মধ্যে সংযোগ তৈরি করে।
\item broadband, DSL, cable বা fiber connection-এর ক্ষেত্রে ভিন্ন ধরনের modem/ONT ব্যবহৃত হতে পারে।
\item internet connection চালু করতে গুরুত্বপূর্ণ device।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{বাস্তব উদাহরণ}
বাসায় broadband internet connection-এ ISP line থেকে signal এনে router বা computer-এ internet দেওয়ার জন্য modem/ONT ব্যবহার করা হয়।
\end{examplebox}

\subsection{Hub}

\begin{definitionbox}{Hub}
Hub হলো একটি basic network device যা একাধিক device-কে LAN-এ যুক্ত করে এবং received data signal সব port-এ পাঠিয়ে দেয়।
\end{definitionbox}

Hub সাধারণত Physical Layer device। এটি intelligent filtering করে না। Hub কোনো data কোন device-এর জন্য তা বুঝতে পারে না; তাই একটি port থেকে data পেলে বাকি সব port-এ পাঠিয়ে দেয়। এজন্য hub-এ traffic বেশি হয় এবং collision-এর সম্ভাবনা বাড়ে।

\begin{keypointbox}{Hub-এর বৈশিষ্ট্য}
\begin{itemize}
\item একাধিক device connect করতে ব্যবহৃত হয়।
\item data সব connected device-এর দিকে পাঠায়।
\item MAC address table ব্যবহার করে না।
\item security ও efficiency কম।
\item আধুনিক network-এ hub-এর পরিবর্তে switch বেশি ব্যবহৃত হয়।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{সহজ উদাহরণ}
একটি পুরনো computer lab-এ ৫টি computer hub দিয়ে যুক্ত করলে, এক computer data পাঠালে hub সেটি অন্য সব computer-এর দিকে পাঠাবে।

\subsection{Switch}

\begin{definitionbox}{Switch}
Switch হলো একটি network device যা LAN-এর একাধিক device-কে যুক্ত করে এবং MAC address ব্যবহার করে data frame নির্দিষ্ট destination device-এর কাছে পাঠায়।
\end{definitionbox}

Switch hub-এর তুলনায় বেশি intelligent। Switch প্রতিটি connected device-এর MAC address শিখে একটি MAC address table তৈরি করে। এরপর data frame আসলে switch table দেখে নির্দিষ্ট port-এ frame পাঠায়। ফলে unnecessary traffic কমে এবং network performance বৃদ্ধি পায়।

\begin{keypointbox}{Switch-এর বৈশিষ্ট্য}
\begin{itemize}
\item LAN-এর device-গুলোকে connect করে।
\item MAC address অনুযায়ী data forwarding করে।
\item hub-এর মতো সব port-এ data পাঠায় না।
\item collision কমায় এবং performance বাড়ায়।
\item computer lab, office network ও campus LAN-এ বহুল ব্যবহৃত।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{বাস্তব উদাহরণ}
School computer lab-এর ৩০টি computer একটি switch-এর সাথে যুক্ত থাকলে, এক PC থেকে printer-এ data গেলে switch নির্দিষ্ট port ব্যবহার করে data পাঠায়।

\subsection{Router}

\begin{definitionbox}{Router}
Router হলো একটি network device যা দুই বা ততোধিক network-কে সংযুক্ত করে এবং IP address ব্যবহার করে data packet-এর সঠিক পথ বা route নির্ধারণ করে।
\end{definitionbox}

Router LAN-কে WAN বা internet-এর সাথে যুক্ত করতে ব্যবহৃত হয়। এটি routing table ব্যবহার করে data packet কোন network path দিয়ে যাবে তা নির্ধারণ করে। বাসার Wi-Fi router সাধারণত LAN device-গুলোকে internet access দেয়।

\begin{keypointbox}{Router-এর বৈশিষ্ট্য}
\begin{itemize}
\item একাধিক network connect করে।
\item IP address ব্যবহার করে packet forwarding করে।
\item routing table ব্যবহার করে best path নির্ধারণ করতে পারে।
\item LAN-কে internet-এর সাথে যুক্ত করে।
\item NAT, DHCP, firewall feature ইত্যাদি অনেক router-এ থাকতে পারে।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{বাস্তব উদাহরণ}
বাসার Wi-Fi router laptop, mobile ও smart TV-কে local network-এ যুক্ত করে এবং modem/ISP-এর মাধ্যমে internet access দেয়।

\subsection{Repeater}

\begin{definitionbox}{Repeater}
Repeater হলো এমন একটি network device যা দুর্বল বা বিকৃত signal গ্রহণ করে পুনরায় শক্তিশালী বা regenerate করে পরবর্তী অংশে পাঠায়।
\end{definitionbox}

দীর্ঘ distance অতিক্রম করলে signal attenuation-এর কারণে দুর্বল হয়ে যায়। Repeater সেই signal নতুন করে তৈরি করে বা শক্তিশালী করে network range বাড়াতে সাহায্য করে। Wired network ও wireless network—দুই ক্ষেত্রেই repeater ধারণা ব্যবহার করা যায়।

\begin{keypointbox}{Repeater-এর কাজ}
\begin{itemize}
\item দুর্বল signal regenerate করে।
\item network distance বা coverage বাড়ায়।
\item Physical Layer-এ কাজ করে।
\item data filtering বা routing করে না।
\item long cable link বা Wi-Fi range extension-এ ব্যবহার করা যায়।
\end{itemize}
\end{keypointbox}

\subsection{Bridge}

\begin{definitionbox}{Bridge}
Bridge হলো একটি network device যা একই ধরনের network-এর দুই বা ততোধিক segment-কে যুক্ত করে এবং MAC address ব্যবহার করে traffic filter করে।
\end{definitionbox}

Bridge LAN segment ভাগ করতে ব্যবহৃত হয়। এটি কোন frame কোন segment-এ পাঠাতে হবে তা MAC address দেখে সিদ্ধান্ত নিতে পারে। ফলে unnecessary traffic কমে এবং network performance উন্নত হয়। Modern switch-কে অনেক সময় multiport bridge হিসেবেও বোঝানো হয়।

\begin{keypointbox}{Bridge-এর বৈশিষ্ট্য}
\begin{itemize}
\item LAN segment connect করে।
\item MAC address দেখে frame filtering করে।
\item unnecessary traffic কমায়।
\item Data Link Layer-এ কাজ করে।
\item ছোট network segment management-এ ব্যবহার করা যায়।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{সহজ উদাহরণ}

\subsection{Gateway}

\begin{definitionbox}{Gateway}
Gateway হলো এমন একটি network device বা node যা ভিন্ন protocol, ভিন্ন architecture বা ভিন্ন ধরনের network-এর মধ্যে data communication সম্ভব করে।
\end{definitionbox}

Gateway-কে protocol converter বলা হয়। Router সাধারণত IP network-এর মধ্যে packet path নির্ধারণ করে, কিন্তু gateway ভিন্ন ধরনের network বা protocol-এর মধ্যে translation করতে পারে। অনেক home network-এ router default gateway হিসেবে কাজ করে।

\begin{keypointbox}{Gateway-এর বৈশিষ্ট্য}
\begin{itemize}
\item ভিন্ন network বা protocol connect করে।
\item protocol conversion করতে পারে।
\item network-এর entry ও exit point হিসেবে কাজ করতে পারে।
\item router-এর তুলনায় বেশি complex হতে পারে।
\item internet access বা enterprise network integration-এ গুরুত্বপূর্ণ।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{বাস্তব উদাহরণ}

\subsection{Firewall}

Firewall unauthorized access, harmful traffic, suspicious connection বা unwanted data block করতে সাহায্য করে। এটি hardware firewall, software firewall বা router-integrated firewall হতে পারে।

\subsection{Common Mistake Zone}

\begin{mistakebox}{Common mistake ১}
অনেকে switch ও router একই মনে করে। Switch একই LAN-এর device-গুলোর মধ্যে data forward করে, কিন্তু router ভিন্ন network connect করে এবং IP address অনুযায়ী route নির্ধারণ করে।
\end{mistakebox}

\begin{mistakebox}{Common mistake ২}
Hub ও switch দেখতে একই রকম মনে হতে পারে, কিন্তু hub data সব port-এ পাঠায়; switch destination MAC address দেখে নির্দিষ্ট port-এ data পাঠায়।
\end{mistakebox}

\begin{mistakebox}{Common mistake ৩}
Modem ও router এক জিনিস নয়। Modem ISP signal convert করে, আর router local network-এর device-গুলোকে network/internet access দেয়।
\end{mistakebox}

\begin{mistakebox}{Common mistake ৪}
Gateway ও router অনেক সময় একই device-এ থাকতে পারে, কিন্তু concept আলাদা। Gateway ভিন্ন protocol বা ভিন্ন network-এর মধ্যে entry/exit point ও translator হিসেবে কাজ করতে পারে।
\end{mistakebox}

\subsection{Exam-style প্রশ্ন ও উত্তর}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Network device কী?}
কম্পিউটার নেটওয়ার্কে device বা network-কে সংযুক্ত করা, data আদান-প্রদান নিয়ন্ত্রণ করা, signal পুনরুদ্ধার করা, route নির্ধারণ করা এবং security নিশ্চিত করার জন্য ব্যবহৃত hardware বা hardware-software সমন্বিত উপাদানকে network device বলা হয়। যেমন—NIC, modem, hub, switch, router, repeater, bridge, gateway, access point ও firewall।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: NIC কী?}
NIC বা Network Interface Card হলো এমন একটি hardware component যার মাধ্যমে computer বা device network-এর সাথে যুক্ত হয়। NIC wired বা wireless হতে পারে এবং device identification-এর জন্য MAC address ব্যবহার করে।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Modem কী?}
Modem হলো এমন একটি network device যা analog signal-কে digital signal-এ এবং digital signal-কে analog signal-এ রূপান্তর করে data communication সম্ভব করে। Modem শব্দটি Modulator ও Demodulator থেকে এসেছে।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Hub কী?}
Hub হলো basic network device যা একাধিক device-কে LAN-এ যুক্ত করে এবং received data সব connected port-এ পাঠিয়ে দেয়। এটি address filtering করে না, তাই efficiency কম।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Switch কী?}
Switch হলো LAN device connecting network device, যা MAC address ব্যবহার করে data frame নির্দিষ্ট destination device-এর port-এ পাঠায়। ফলে unnecessary traffic কমে এবং network performance বাড়ে।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Router কী?}
Router হলো network device যা দুই বা ততোধিক network-কে সংযুক্ত করে এবং IP address ব্যবহার করে data packet-এর সঠিক route নির্ধারণ করে। LAN-কে internet-এর সাথে যুক্ত করতে router ব্যবহৃত হয়।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Repeater কী?}
Repeater হলো network device যা দুর্বল signal গ্রহণ করে পুনরায় শক্তিশালী বা regenerate করে পরবর্তী অংশে পাঠায়। এটি network range বাড়াতে ব্যবহৃত হয়।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Bridge কী?}
Bridge হলো network device যা একই ধরনের network-এর দুই বা ততোধিক LAN segment যুক্ত করে এবং MAC address অনুযায়ী traffic filter করে।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Gateway কী?}
Gateway হলো network device বা node যা ভিন্ন protocol, ভিন্ন architecture বা ভিন্ন ধরনের network-এর মধ্যে data communication সম্ভব করে। Gateway-কে protocol converter-ও বলা হয়।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Access Point কী?}
Access Point হলো network device যা wireless device-গুলোকে wired network বা LAN-এর সাথে যুক্ত হওয়ার সুযোগ দেয়। এটি Wi-Fi signal তৈরি করে mobile, laptop ও tablet-কে network access দেয়।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Firewall কী?}
Firewall হলো network security device বা software system যা predefined security rules অনুযায়ী incoming ও outgoing network traffic monitor এবং filter করে। এটি unauthorized access ও harmful traffic থেকে network রক্ষা করতে সাহায্য করে।
\end{boardbox}

\begin{boardbox}{পার্থক্য: Hub ও Switch}
Hub received data সব port-এ পাঠায় এবং address filtering করে না। তাই traffic ও collision বেশি হতে পারে। অন্যদিকে switch MAC address table ব্যবহার করে data নির্দিষ্ট destination port-এ পাঠায়। তাই switch বেশি efficient এবং আধুনিক LAN-এ hub-এর পরিবর্তে switch ব্যবহার করা হয়।
\end{boardbox}

\begin{boardbox}{পার্থক্য: Switch ও Router}
Switch একই LAN-এর একাধিক device connect করে এবং MAC address অনুযায়ী frame forward করে। Router দুই বা ততোধিক network connect করে এবং IP address অনুযায়ী packet route নির্ধারণ করে। তাই switch LAN-এর ভিতরে বেশি ব্যবহৃত হয়, আর router LAN-কে internet বা অন্য network-এর সাথে যুক্ত করে।
\end{boardbox}
