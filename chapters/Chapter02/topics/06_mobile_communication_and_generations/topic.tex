Mobile communication এমন একটি ব্যবস্থা, যেখানে ব্যবহারকারী চলমান অবস্থাতেও voice, message, data ও multimedia service পেতে পারে। এর ভিত্তি হলো cellular network।

\begin{definitionbox}{Mobile communication}
মোবাইল ডিভাইস ব্যবহার করে চলমান অবস্থায় voice, data এবং multimedia আদান-প্রদানের প্রযুক্তিকে mobile communication বলা হয়।
\end{definitionbox}

\subsection{Cellular Concept}

Cellular network-এ একটি বড় coverage area ছোট ছোট cell-এ ভাগ করা হয়। প্রতিটি cell-এ base station থাকে এবং একই frequency পুনর্ব্যবহার করা যায়। ফলে বেশি ব্যবহারকারীকে একই সাথে সেবা দেওয়া সম্ভব হয়।

\begin{keypointbox}{Cellular network-এর অংশ}
\begin{itemize}
\item Cell: coverage-এর ছোট area
\item Base station: signal পাঠানো ও গ্রহণের কেন্দ্র
\item Mobile switching center: connection control ও handover management
\item Frequency reuse: channel দক্ষতার সঙ্গে পুনরায় ব্যবহার
\end{itemize}
\end{keypointbox}

\begin{examplebox}{Cellular ধারণা}
একটি শহরকে অনেক cell-এ ভাগ করলে একই frequency আলাদা cell-এ ব্যবহৃত হতে পারে। এতে network capacity বাড়ে এবং spectrum waste কমে।
\end{examplebox}

\subsection{Generations of Mobile Communication}

\begin{center}
\begin{tabular}{|p{0.13\textwidth}|p{0.24\textwidth}|p{0.43\textwidth}|}
\hline
\textbf{Gen} & \textbf{Type} & \textbf{মূল বৈশিষ্ট্য} \\
\hline

\hline

\begin{definitionbox}{5G}
5G হলো mobile communication-এর এমন advanced generation, যা খুব দ্রুত data transfer, কম latency এবং massive device connectivity সমর্থন করে।
\end{definitionbox}

\begin{keypointbox}{Generation trend}
\begin{itemize}
\item voice থেকে data-এর দিকে অগ্রগতি হয়েছে।
\item speed ক্রমে বেড়েছে।
\item security ও capacity উন্নত হয়েছে।
\item multimedia ও IoT support বেড়েছে।
\end{itemize}
\end{keypointbox}

\subsection{Mobile Network Basics}

\begin{definitionbox}{Handover}
চলমান user এক cell থেকে অন্য cell-এ গেলে call বা data session না ভেঙে connection transfer করার প্রক্রিয়াকে handover বলা হয়।
\end{definitionbox}

\begin{definitionbox}{Roaming}
নিজ network coverage-এর বাইরে অন্য network-এর coverage ব্যবহার করে service গ্রহণ করার সক্ষমতাকে roaming বলে।
\end{definitionbox}

\begin{examplebox}{SIM and authentication}
SIM user identity বহন করে, আর authentication নিশ্চিত করে যে network-এ সঠিক subscriber যুক্ত হচ্ছে কি না।
\end{examplebox}

\begin{boardbox}{বোর্ড ফোকাস}
1G to 5G প্রশ্নে শুধু speed না লিখে voice, data, latency, security ও application উল্লেখ করতে হয়। Handover ও roaming definition-সহ example দিলে উত্তর শক্তিশালী হয়।
\end{boardbox}

\begin{recapbox}
