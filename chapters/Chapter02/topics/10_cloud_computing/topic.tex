\section{Cloud Computing}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী পারবে:
\begin{itemize}
\item cloud computing-এর মৌলিক ধারণা বলতে।
\item on-demand resource, scalability এবং internet-based service ব্যাখ্যা করতে।
\item IaaS, PaaS ও SaaS compare করতে।
\item public, private, hybrid ও community cloud আলাদা করতে।
\item cloud service model-এর tree diagram লিখতে।
\end{itemize}
\end{learningbox}

Cloud computing হলো এমন একটি model, যেখানে local device-এ সব resource না রেখে internet-এর মাধ্যমে remote server থেকে storage, software, processing ও platform ব্যবহার করা হয়। এতে cost কমে, flexibility বাড়ে এবং collaboration সহজ হয়।

\begin{definitionbox}{Cloud computing}
ইন্টারনেটের মাধ্যমে demand অনুযায়ী computing resource, storage, software এবং platform ব্যবহার করার পদ্ধতিকে cloud computing বলা হয়।
\end{definitionbox}

\begin{keypointbox}{Cloud computing-এর বৈশিষ্ট্য}
\begin{itemize}
\item on-demand access
\item scalability
\item resource pooling
\item pay-as-you-go model
\item device-independent access
\end{itemize}
\end{keypointbox}

\begin{examplebox}{উদাহরণ}
Cloud storage IaaS-এর কাছাকাছি service দিতে পারে, developer platform PaaS-এর উদাহরণ, আর Gmail, Google Docs, Office 365 ধরনের service SaaS-এর উদাহরণ।
\end{examplebox}

অনলাইন class platform, document sharing, backup storage এবং software subscription cloud computing-এর বাস্তব উদাহরণ। এ ধরনের service remote work, freelancing এবং institutional collaboration সহজ করে।
\end{examplebox}

\begin{boardbox}{বোর্ড ফোকাস}
Cloud computing প্রশ্নে definition-এর পরে service model আর deployment model আলাদা করে লিখতে হয়। IaaS, PaaS, SaaS গুলোর role mix না করে লিখতে হবে।
\end{boardbox}

\begin{recapbox}
\begin{itemize}
\item Cloud computing internet-based resource access।
\item IaaS infrastructure, PaaS platform, SaaS software দেয়।
\item Public, private, hybrid ও community deployment model আছে।
\item On-demand এবং scalable হওয়াই cloud-এর বড় শক্তি।
\end{itemize}
\end{recapbox}

\begin{keypointbox}{Cloud-এর সুবিধা}
\begin{itemize}
\item প্রয়োজন অনুযায়ী resource বাড়ানো-কমানো যায়।
\item শুরুতে hardware কেনার খরচ কমে।
\item internet থাকলে যেকোনো জায়গা থেকে access করা যায়।
\item backup, collaboration ও maintenance সহজ হয়।
\end{itemize}
\end{keypointbox}

\begin{mistakebox}{সাধারণ ভুল}
Cloud মানে শুধু online storage নয়। Cloud computing-এ server, storage, platform, software, network resource—সবই service হিসেবে পাওয়া যায়।
\end{mistakebox}
