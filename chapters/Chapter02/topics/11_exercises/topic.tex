\subsection{দ্রুত উত্তর-ইঙ্গিত}
\begin{itemize}
\item Bandwidth: প্রতি সেকেন্ডে channel দিয়ে data বহনের ক্ষমতা।
\item Asynchronous: character-by-character, start/stop bit থাকে।
\item Synchronous: block-by-block, timing synchronized।
\item Simplex: একমুখী; Half duplex: দুইমুখী কিন্তু একসাথে নয়; Full duplex: দুইমুখী একসাথে।
\item Guided media: cable নির্ভর; Unguided media: electromagnetic wave নির্ভর।
\item Router ভিন্ন network যুক্ত করে; Switch LAN-এর মধ্যে নির্দিষ্ট device-এ frame পাঠায়।
\end{itemize}
