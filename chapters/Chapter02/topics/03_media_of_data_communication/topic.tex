\section{ডেটা কমিউনিকেশন মাধ্যম}

\begin{learningbox}
এই টপিক শেষে শিক্ষার্থী পারবে:
\begin{itemize}
\item ডেটা কমিউনিকেশন মাধ্যম বা transmission medium কী তা ব্যাখ্যা করতে।
\item Guided/Wired media এবং Unguided/Wireless media-এর পার্থক্য বুঝতে।
\item Twisted pair, Coaxial cable, Optical fiber, Radio wave, Microwave ও Infrared-এর ধারণা নিতে।
\item মাধ্যম নির্বাচনের ক্ষেত্রে bandwidth, cost, distance, security, noise ও installation factor বিশ্লেষণ করতে।
\item বাস্তব জীবনের উদাহরণ থেকে উপযুক্ত communication medium শনাক্ত করতে।
\item বোর্ড ও ভর্তি পরীক্ষার জন্য medium-based comparison answer লিখতে।
\end{itemize}
\end{learningbox}

\subsection{বাস্তব জীবনের শুরু}

একজন শিক্ষার্থী যখন Wi-Fi দিয়ে online class করে, তখন data বাতাসের মাধ্যমে electromagnetic wave আকারে router পর্যন্ত যায়। আবার router থেকে ISP network-এ data যেতে পারে fiber optic cable বা অন্য কোনো cable দিয়ে। অর্থাৎ data এক device থেকে অন্য device-এ যেতে একটি পথ বা মাধ্যম প্রয়োজন হয়।

এই পথ বা মাধ্যমই হলো \textbf{ডেটা কমিউনিকেশন মাধ্যম}। Medium ছাড়া sender-এর data receiver-এর কাছে পৌঁছাতে পারে না।

\begin{keypointbox}{মূল ধারণা}
Data communication medium বুঝতে তিনটি প্রশ্ন মনে রাখো:
\begin{itemize}
\item data কোন পথে যাচ্ছে?
\item পথটি cable-based, নাকি wireless?
\item medium-এর speed, distance, cost ও security কেমন?
\end{itemize}
\end{keypointbox}

\subsection{ডেটা কমিউনিকেশন মাধ্যম}

\begin{definitionbox}{ডেটা কমিউনিকেশন মাধ্যম}
Sender থেকে receiver-এর কাছে data বা signal পাঠানোর জন্য যে physical বা wireless path ব্যবহৃত হয়, তাকে data communication medium বা transmission medium বলা হয়।
\end{definitionbox}

Communication system-এ sender message তৈরি করে এবং transmitter সেই message-কে signal-এ রূপান্তর করে। এরপর signal কোনো medium দিয়ে receiver-এর কাছে যায়। এই medium হতে পারে copper cable, fiber optic cable, radio wave, microwave, infrared ray বা satellite link।

\begin{examplebox}{সহজ উদাহরণ}
\begin{itemize}
\item Computer lab-এ PC থেকে switch-এ data গেলে twisted pair cable medium হিসেবে কাজ করে।
\item Broadband internet-এ ISP থেকে বাসা পর্যন্ত fiber optic cable medium হিসেবে কাজ করতে পারে।
\item Mobile phone থেকে tower-এ signal গেলে radio wave medium হিসেবে কাজ করে।
\item TV remote থেকে TV-তে signal গেলে infrared medium হিসেবে কাজ করে।
\end{itemize}
\end{examplebox}

\subsection{Transmission Medium-এর গুরুত্ব}

Communication medium network performance-এর উপর সরাসরি প্রভাব ফেলে। একই data যদি নিম্নমানের medium দিয়ে পাঠানো হয়, তাহলে speed কমতে পারে, noise বাড়তে পারে, error হতে পারে এবং data loss ঘটতে পারে। তাই network design করার সময় medium নির্বাচন খুব গুরুত্বপূর্ণ।

\begin{keypointbox}{Medium গুরুত্বপূর্ণ কেন?}
\begin{itemize}
\item data কত দ্রুত যাবে তা medium-এর bandwidth-এর উপর নির্ভর করে।
\item data কত দূর যাবে তা medium-এর attenuation ও range-এর উপর নির্ভর করে।
\item data কতটা নিরাপদ থাকবে তা medium-এর security ও signal leakage-এর উপর নির্ভর করে।
\item network installation cost medium-এর ধরন অনুযায়ী পরিবর্তিত হয়।
\item noise, interference ও weather condition কিছু medium-কে বেশি প্রভাবিত করতে পারে।
\end{itemize}
\end{keypointbox}

\subsection{Guided Media বা Wired Media}

\begin{definitionbox}{Guided Media}
যে communication medium-এ signal কোনো নির্দিষ্ট physical path বা cable-এর মাধ্যমে sender থেকে receiver-এর কাছে যায়, তাকে guided media বা wired media বলা হয়।
\end{definitionbox}

Guided media-তে signal নির্দিষ্ট cable-এর ভিতর দিয়ে যায়। তাই signal path controlled থাকে এবং security তুলনামূলক বেশি হয়। তবে cable বসাতে installation cost ও সময় লাগে।

\begin{keypointbox}{Guided media-এর বৈশিষ্ট্য}
\begin{itemize}
\item physical cable বা wire ব্যবহার করা হয়।
\item signal নির্দিষ্ট path দিয়ে যায়।
\item সাধারণত security বেশি।
\item interference তুলনামূলক কম হতে পারে।
\item installation করতে cable laying দরকার হয়।
\item building, office, school lab, data center ও backbone network-এ বেশি ব্যবহৃত হয়।
\end{itemize}
\end{keypointbox}

\subsection{Twisted Pair Cable}

\begin{definitionbox}{Twisted Pair Cable}
দুটি insulated copper wire পরস্পর পেঁচিয়ে তৈরি cable-কে twisted pair cable বলা হয়।
\end{definitionbox}

Twisted pair cable-এ দুটি copper wire পরস্পর twist করা থাকে। এই twisting electromagnetic interference কমাতে সাহায্য করে। Local Area Network বা LAN-এ এটি খুব বেশি ব্যবহৃত হয়। সাধারণ office, school lab, computer lab, router-to-PC connection ইত্যাদিতে twisted pair cable দেখা যায়।

\begin{keypointbox}{Twisted pair cable-এর বৈশিষ্ট্য}
\begin{itemize}
\item copper wire দিয়ে তৈরি।
\item installation সহজ এবং cost কম।
\item LAN connection-এ ব্যাপকভাবে ব্যবহৃত।
\item distance বেশি হলে signal দুর্বল হতে পারে।
\item electromagnetic interference দ্বারা প্রভাবিত হতে পারে।
\item UTP এবং STP—এই দুই ধরনের হতে পারে।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{বাস্তব উদাহরণ}
বাড়ির router থেকে computer বা office switch থেকে desktop PC-তে যে Ethernet cable ব্যবহার করা হয়, সেটি সাধারণত twisted pair cable।
\end{examplebox}

\subsection{Coaxial Cable}

\begin{definitionbox}{Coaxial Cable}
একটি central copper conductor, insulating layer, metallic shield এবং outer jacket নিয়ে গঠিত cable-কে coaxial cable বলা হয়।
\end{definitionbox}

Coaxial cable-এ মাঝখানে copper conductor থাকে। এর চারপাশে insulation, metallic shield এবং outer cover থাকে। Metallic shield external noise থেকে signal-কে কিছুটা সুরক্ষা দেয়। Cable TV, broadband cable network এবং পুরনো Ethernet network-এ coaxial cable ব্যবহৃত হয়েছে।

\begin{keypointbox}{Coaxial cable-এর বৈশিষ্ট্য}
\begin{itemize}
\item central conductor দিয়ে signal যায়।
\item metallic shield noise কমাতে সাহায্য করে।
\item twisted pair-এর তুলনায় interference কম হতে পারে।
\item cable TV network-এ বহুল ব্যবহৃত।
\item installation twisted pair-এর তুলনায় কিছুটা কঠিন হতে পারে।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{বাস্তব উদাহরণ}
Cable TV line বা পুরনো broadband cable connection-এ coaxial cable ব্যবহৃত হতে দেখা যায়।
\end{examplebox}

\subsection{Optical Fiber Cable}

\begin{definitionbox}{Optical Fiber Cable}
যে cable-এ glass বা plastic fiber-এর মাধ্যমে light signal ব্যবহার করে data পাঠানো হয়, তাকে optical fiber cable বলা হয়।
\end{definitionbox}

Optical fiber cable modern communication system-এর অন্যতম গুরুত্বপূর্ণ medium। এতে electrical signal নয়, light signal ব্যবহৃত হয়। ফলে high bandwidth, long distance এবং low interference পাওয়া যায়। Internet backbone, submarine cable, FTTH broadband এবং data center connection-এ optical fiber খুব গুরুত্বপূর্ণ।

\begin{keypointbox}{Optical fiber cable-এর বৈশিষ্ট্য}
\begin{itemize}
\item light signal ব্যবহার করে data পাঠায়।
\item bandwidth খুব বেশি।
\item long distance communication-এর জন্য উপযোগী।
\item electromagnetic interference দ্বারা প্রভাবিত হয় না।
\item security তুলনামূলক বেশি।
\item installation ও maintenance cost তুলনামূলক বেশি।
\item cable ভেঙে গেলে repair করা তুলনামূলক কঠিন।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{বাস্তব উদাহরণ}
FTTH broadband, intercity internet backbone, submarine cable এবং data center connection-এ optical fiber ব্যবহার করা হয়।
\end{examplebox}

\subsection{Unguided Media বা Wireless Media}

\begin{definitionbox}{Unguided Media}
যে communication medium-এ signal কোনো physical cable ছাড়াই বাতাস, মহাকাশ বা free space-এর মাধ্যমে electromagnetic wave আকারে sender থেকে receiver-এর কাছে যায়, তাকে unguided media বা wireless media বলা হয়।
\end{definitionbox}

Wireless media-তে cable লাগে না। ফলে mobility বেশি এবং installation তুলনামূলক সহজ। তবে wireless signal interference, obstacle, distance, weather ও security risk দ্বারা বেশি প্রভাবিত হতে পারে।

\begin{keypointbox}{Unguided media-এর বৈশিষ্ট্য}
\begin{itemize}
\item physical cable ছাড়াই data transmission হয়।
\item electromagnetic wave ব্যবহার করা হয়।
\item mobility বেশি।
\item installation flexible।
\item interference ও obstacle দ্বারা প্রভাবিত হতে পারে।
\item security নিশ্চিত করতে encryption প্রয়োজন।
\end{itemize}
\end{keypointbox}

\subsection{Radio Wave}

\begin{definitionbox}{Radio Wave}
যে electromagnetic wave তুলনামূলকভাবে দীর্ঘ wavelength ব্যবহার করে wireless communication-এ data বা signal বহন করে, তাকে radio wave বলা হয়।
\end{definitionbox}

Radio wave সাধারণত সহজে চারদিকে ছড়িয়ে পড়ে এবং obstacle কিছুটা অতিক্রম করতে পারে। Radio broadcasting, mobile communication, Wi-Fi, Bluetooth ইত্যাদির সাথে radio wave-এর ধারণা সম্পর্কিত।

\begin{keypointbox}{Radio wave-এর বৈশিষ্ট্য}
\begin{itemize}
\item wireless communication-এ ব্যবহৃত হয়।
\item তুলনামূলকভাবে বড় area cover করতে পারে।
\item obstacle কিছুটা অতিক্রম করতে পারে।
\item antenna ব্যবহার করা হয়।
\item interference ও noise দ্বারা প্রভাবিত হতে পারে।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{বাস্তব উদাহরণ}
FM radio, mobile phone communication, Wi-Fi signal ও Bluetooth communication—radio wave-এর বাস্তব ব্যবহার।
\end{examplebox}

\subsection{Microwave}

\begin{definitionbox}{Microwave}
উচ্চ frequency-এর electromagnetic wave, যা point-to-point wireless communication-এ ব্যবহৃত হয়, তাকে microwave বলা হয়।
\end{definitionbox}

Microwave communication সাধারণত line-of-sight transmission-এর উপর নির্ভর করে। অর্থাৎ transmitter ও receiver-এর মধ্যে সরাসরি path থাকা দরকার। Tower-to-tower communication, satellite communication, cellular backbone ইত্যাদিতে microwave ব্যবহৃত হতে পারে।

\begin{keypointbox}{Microwave-এর বৈশিষ্ট্য}
\begin{itemize}
\item উচ্চ frequency electromagnetic wave ব্যবহার করে।
\item point-to-point communication-এ ব্যবহৃত হয়।
\item line-of-sight path দরকার হতে পারে।
\item long distance communication-এ tower বা repeater ব্যবহার করা হয়।
\item weather condition ও obstacle দ্বারা প্রভাবিত হতে পারে।
\end{itemize}
\end{keypointbox}

\begin{examplebox}{বাস্তব উদাহরণ}
Mobile tower-to-tower link, satellite link এবং long-distance wireless backbone communication-এ microwave ব্যবহৃত হয়।
\end{examplebox}

\subsection{Infrared}

\begin{definitionbox}{Infrared}
Infrared হলো এমন electromagnetic radiation যা short-range wireless communication-এ ব্যবহৃত হয় এবং সাধারণত line-of-sight প্রয়োজন হয়।
\end{definitionbox}

Infrared signal সাধারণত wall বা obstacle অতিক্রম করতে পারে না। তাই এটি short distance communication-এ ব্যবহৃত হয়। TV remote control infrared-এর খুব পরিচিত উদাহরণ।

\begin{keypointbox}{Infrared-এর বৈশিষ্ট্য}
\begin{itemize}
\item short-range communication-এ ব্যবহৃত হয়।
\item line-of-sight দরকার।
\item wall বা obstacle অতিক্রম করতে পারে না।
\item cost কম এবং সহজ ব্যবহারযোগ্য।
\item TV remote, AC remote ইত্যাদিতে ব্যবহৃত হয়।
\end{itemize}
\end{keypointbox}

\subsection{Satellite Communication}

\begin{definitionbox}{Satellite Communication}
যে communication system-এ earth station থেকে signal satellite-এ পাঠানো হয় এবং satellite সেই signal অন্য earth station-এ retransmit করে, তাকে satellite communication বলা হয়।
\end{definitionbox}

Satellite communication মূলত microwave signal ব্যবহার করে। এটি বড় area coverage, international communication, weather forecasting, TV broadcasting, GPS এবং remote area communication-এর জন্য গুরুত্বপূর্ণ।

\subsection{Transmission Medium নির্বাচনের বিষয়গুলো}

সব network-এর জন্য একই medium best নয়। কোথায় কোন medium ব্যবহার হবে তা নির্ভর করে ব্যবহার, cost, distance, bandwidth, security এবং environment-এর উপর।

\begin{keypointbox}{Medium নির্বাচন করার সময় বিবেচ্য বিষয়}
\begin{itemize}
\item \textbf{Bandwidth}: কত বেশি data বহন করতে হবে?
\item \textbf{Distance}: কত দূর পর্যন্ত communication দরকার?
\item \textbf{Cost}: installation ও maintenance cost কত?
\item \textbf{Security}: data কতটা sensitive?
\item \textbf{Noise/Interference}: environment-এ electromagnetic noise আছে কি?
\item \textbf{Mobility}: user চলাচল করবে কি না?
\item \textbf{Reliability}: connection কতটা stable দরকার?
\item \textbf{Scalability}: ভবিষ্যতে network বড় করা লাগবে কি না?
\end{itemize}
\end{keypointbox}

\subsection{Common Mistake Zone}

\begin{mistakebox}{Common mistake ১}
অনেকে medium ও mode গুলিয়ে ফেলে। Medium হলো data যাওয়ার পথ, যেমন cable বা radio wave। Mode হলো data কোন direction-এ যাচ্ছে, যেমন simplex, half-duplex, full-duplex।
\end{mistakebox}

\begin{mistakebox}{Common mistake ২}
Wireless মানেই সবসময় faster—এটি সঠিক নয়। Wireless flexible হলেও interference, distance, obstacle ও signal strength speed কমাতে পারে।
\end{mistakebox}

\begin{mistakebox}{Common mistake ৩}
Fiber optic cable-কে অনেকেই electric signal বহন করে মনে করে। আসলে fiber optic cable light signal ব্যবহার করে data পাঠায়।
\end{mistakebox}

\subsection{Exam-style প্রশ্ন ও উত্তর}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: ডেটা কমিউনিকেশন মাধ্যম কী?}
Sender থেকে receiver-এর কাছে data বা signal পাঠানোর জন্য যে physical বা wireless path ব্যবহৃত হয়, তাকে data communication medium বা transmission medium বলা হয়। যেমন—twisted pair cable, coaxial cable, optical fiber cable, radio wave, microwave ইত্যাদি।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Guided media কী?}
যে communication medium-এ signal নির্দিষ্ট physical cable বা wire-এর মাধ্যমে sender থেকে receiver-এর কাছে যায়, তাকে guided media বা wired media বলা হয়। যেমন—twisted pair cable, coaxial cable ও optical fiber cable।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Unguided media কী?}
যে communication medium-এ signal physical cable ছাড়াই বাতাস, মহাকাশ বা free space দিয়ে electromagnetic wave আকারে পাঠানো হয়, তাকে unguided media বা wireless media বলা হয়। যেমন—radio wave, microwave, infrared ও satellite link।
\end{boardbox}

\begin{boardbox}{পার্থক্য: Guided media ও Unguided media}
Guided media-তে signal নির্দিষ্ট cable বা physical path দিয়ে যায়, কিন্তু unguided media-তে signal বাতাস বা space দিয়ে electromagnetic wave আকারে যায়। Guided media-এর security তুলনামূলক বেশি, কিন্তু mobility কম। Unguided media mobility বেশি দেয়, তবে interference ও security risk বেশি হতে পারে। Twisted pair, coaxial ও fiber optic guided media-এর উদাহরণ; radio wave, microwave ও infrared unguided media-এর উদাহরণ।
\end{boardbox}

\begin{boardbox}{সংক্ষিপ্ত প্রশ্ন: Optical fiber কেন high-speed communication-এ ব্যবহৃত হয়?}
Optical fiber cable light signal ব্যবহার করে data পাঠায়। এতে bandwidth খুব বেশি, long distance communication সম্ভব এবং electromagnetic interference-এর প্রভাব নেই। তাই internet backbone, FTTH broadband ও data center communication-এ optical fiber ব্যবহৃত হয়।
\end{boardbox}

\item Microwave point-to-point এবং satellite communication-এ গুরুত্বপূর্ণ।
